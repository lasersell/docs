Rust SDKの完全なインストール手順、モジュール一覧、APIクライアント使用法、Exit Intelligence Streamセッション、流動性スナップショット、トランザクションヘルパー、エラー処理については、[英語版ドキュメント](/api/sdk/rust)を参照してください。

## インストール

```bash
cargo add lasersell-sdk
cargo add tokio --features full
```
