TypeScript SDKの完全なインストール手順、モジュール一覧、APIクライアント使用法、Exit Intelligence Streamセッション、レーンスプリッティング、流動性スナップショット、トランザクションヘルパー、エラー処理については、[英語版ドキュメント](/api/sdk/typescript)を参照してください。

## インストール

```bash
npm install @lasersell/lasersell-sdk
```

パッケージはTypeScript宣言を同梱し、ESMとCommonJSの両方をサポートしています。

完全な動作例については[クイックスタート](/api/quickstart)を参照してください。
