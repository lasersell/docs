## 対応プロトコル

| `market_type` | プロトコル | タイプ | 説明 |
|----------------------|-----------------------|-----------------|-------------------------------------------------|
| `pump_fun`           | Pump.fun              | ボンディングカーブ | オリジナルのPump.funボンディングカーブ。 |
| `pump_swap`          | PumpSwap              | AMM | 卒業したPump.funトークンのPumpSwap AMM。 |
| `meteora_dbc`        | Meteora DBC           | ダイナミックボンディング | Meteoraダイナミックボンディングカーブプール。 |
| `meteora_damm_v2`    | Meteora DAMM V2       | AMM | Meteora Dynamic AMM V2プール。 |
| `raydium_launchpad`  | Raydium Launchpad     | ボンディングカーブ | Raydiumローンチパッドボンディングカーブトークン。 |
| `raydium_cpmm`       | Raydium CPMM          | AMM | Raydium定積市場メーカープール。 |

## 卒業

| `meteora_dbc`        | `meteora_damm_v2`       |
