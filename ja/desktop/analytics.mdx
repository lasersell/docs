## 概要

期間（24h、7d、30d、またはAll Time）を選択して主要メトリクスを表示:

- **Total P&L** — 表示通貨での正味損益
- **Total Volume** — クォート単位での合計取引量
- **Win Rate** — 利益で終了した取引のパーセンテージ
- **Total Trades** — クローズドポジションの数
- **Avg P&L** — トレードあたりの平均損益
- **Avg Hold Time** — ポジションが開かれていた平均時間
- **Best Trade** — 単一取引での最高利益
- **Worst Trade** — 単一取引での最大損失

### デイリーPnLチャート

過去90日間の日次損益を示す棒グラフ。緑のバーは利益の出た日、赤のバーは損失の日を表します。任意のバーにホバーすると正確な数値が表示されます。

### エグジット理由の内訳

エグジットのトリガーの視覚的な内訳:

- **Target Profit** — 戦略がテイクプロフィット閾値に達した
- **Trailing Stop** — 価格がピークからトレーリングパーセンテージ分下落した
- **Stop Loss** — 価格がストップロス閾値に達した
- **Graduation** — トークンがボンディングカーブからDEXに移行した
- **Deadline** — ポジションがタイムアウトした

これにより、どのエグジット条件が最も機能しているか、戦略の調整が必要かを理解できます。

## インサイト（Tier 1+）

最近の取引のパターンに基づくパーソナライズされた提案。各インサイトには重要度レベル（high、medium、low）と具体的な推奨事項があります。

インサイトは取引履歴からサーバー側で生成され、定期的に更新されます。Professionalティア以上にゲートされています。

Freeティアの場合、アップグレードプロンプトが表示されます。

## アチーブメント

CommonからLegendaryまでのレアリティティアを持つ**10カテゴリ**にわたる進捗を追跡します。

### カテゴリ

| カテゴリ | 追跡内容 |
| :--- | :--- |
| Volume | 取引量マイルストーン |
| Trades | 完了した取引数 |
| Streak | 連続利益取引日 |
| Profit | 累積利益目標 |
| Speed | 高速実行時間 |
| Single Trade | 単一取引PnLレコード |
| Win Rate | 勝率パーセンテージマイルストーン |
| Mirror | コピートレーディングマイルストーン |
| Secret | 隠しアチーブメント（取引で発見） |
| Mastery | 上級組み合わせアチーブメント |

### レアリティティア

- **Common** — 入門レベルのマイルストーン
- **Uncommon** — 中程度の達成
- **Rare** — 重要なアチーブメント
- **Epic** — 例外的な取引マイルストーン
- **Legendary** — システム内で最も希少なアチーブメント

### ストリーク

現在と最長のウィニングストリークがアチーブメントタブの上部に表示されます。LaserSellプロトコルを通じてルーティングされた利益のある取引が少なくとも1回ある日がストリーク日としてカウントされます。

### アチーブメントの仕組み

アチーブメントはサーバーによって毎時チェックされます。LaserSellプロトコルを通じてルーティングされた取引のみがアチーブメントの進捗にカウントされます。新しいアチーブメントをアンロックすると、アプリが通知を送信します。

<Note>
アンロックしたアチーブメントは、将来報酬と交換可能になります。LaserSellトークンは存在せず、エアドロップも行われません。
</Note>
