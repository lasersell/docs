---
title: "アチーブメント"
description: "CommonからLegendaryまでのレアリティティアを持つ10カテゴリにわたる取引マイルストーンを追跡します。"
---

LaserSellはあなたの取引マイルストーンを追跡し、達成するとアチーブメントを授与します。10カテゴリにわたって60以上のアチーブメントがあり、誰でも達成できるCommonマイルストーンから、卓越したパフォーマンスを必要とするLegendaryアチーブメントまで多岐にわたります。

<Note>
アンロックしたアチーブメントは、トレーダーにこれまで提供されたことのない報酬と交換可能になります。LaserSellトークンは存在せず、エアドロップも行われません。
</Note>

## アチーブメントの仕組み

アチーブメントは1時間ごとのサイクルで自動的にチェックされます。LaserSellプロトコルを通じてルーティングされた取引のみがアチーブメントの進捗にカウントされます。新しいアチーブメントをアンロックすると、アプリが通知を送信します。

隠しアチーブメントは、アンロックするまで条件が明かされません。

## レアリティティア

| レアリティ | 説明 |
| :--- | :--- |
| **Common** | 入門レベルのマイルストーン。取引を始めると自然にアンロックされます。 |
| **Uncommon** | 継続的な取引活動を必要とする中程度の達成。 |
| **Rare** | 真の献身やスキルを必要とする重要なアチーブメント。 |
| **Epic** | 少数のトレーダーだけが到達する例外的なマイルストーン。 |
| **Legendary** | システム内で最も希少なアチーブメント。驚異的な取引量、一貫性、または単一の卓越したパフォーマンスの瞬間を必要とします。 |

## カテゴリ

### Volume

累積取引量マイルストーン。LaserSellを通じてより多く取引するほど、上に登っていきます。

### Trades

初回取引から数千回まで、完了した取引の合計マイルストーン。

### Streak

連続取引日数マイルストーン。LaserSellを通じてルーティングされた取引を少なくとも1回完了した日がストリーク日としてカウントされます。1日でも欠けると現在のストリークがリセットされます。

現在のストリークと最長ストリークは、アプリのアチーブメントタブの上部に表示されます。

### Profit

SOLでの累積実現利益マイルストーン。

### Speed

短時間で完了した利益のある取引に対するアチーブメント。素早い実行と迅速なエグジットが報われます。

### Single Trade

堅実なリターンから驚異的なマルチプライヤーまで、単一ポジションのパフォーマンスに基づくアチーブメント。

### Win Rate

ゲーミング防止のために最低取引要件付きの勝率マイルストーン。数百の取引にわたって高い勝率を維持することは、最も難しいことの一つです。

### Mirror

[ミラートレーディング](/desktop/mirror-trading)のアチーブメント。ウォッチウォレットにわたるコピートレーディング活動とパフォーマンスを追跡します。

### Secret

アンロックするまで条件が明かされない隠しアチーブメント。取引を続ける中で自然と発見するでしょう。

### Mastery

カテゴリ内のすべてのアチーブメントを完了することで授与されるメタアチーブメント。他のカテゴリそれぞれに1つのMasteryアチーブメントがあります。すべてのSecretアチーブメントを完了すると、唯一のLegendaryティアMasteryが得られます。

## ストリーク

取引ストリークは、LaserSellを通じてルーティングされた取引を少なくとも1回完了した連続日数を追跡します。

- **Current Streak** — 今日を含めて何日連続で取引しているか。
- **Longest Streak** — 全期間の記録。
- **Last Trade Date** — 最新の取引が記録された日時。

1日でも欠けると現在のストリークはゼロにリセットされます。最長ストリークは永久に保持されます。

## APIアクセス

アチーブメントはプログラムで利用できます:

```
GET /v1/achievements
```

すべてのアチーブメントとそのアンロック状態およびタイムスタンプを返します。Secretカテゴリのロックされたアチーブメントはマスクされた説明を返します。
