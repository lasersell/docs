LaserSellは、プロトコルを通じて行ったすべての取引を追跡し、実用的なパフォーマンスデータに変換します。アナリティクスダッシュボードは、[デスクトップアプリ](/desktop/analytics)およびAPIを通じて利用できます。

## 期間

すべてのメトリクスは4つの期間で表示できます:

- **24時間** — 当日のパフォーマンス
- **7日間** — 週間スナップショット
- **30日間** — 月間概要
- **全期間** — 完全な取引履歴

各期間は独立したメトリクスセットを計算するため、取引がどのように推移してきたかを比較できます。

## 主要メトリクス

### 合計P&L

選択した期間のすべてのクローズドポジションにわたる、正味の実現損益です。すべての取引の最終利益の合計であり、選択した表示通貨で表示されます。

### 合計取引量

期間中のすべてのエントリー金額（トークン購入に使用したSOLまたはUSD1）の合計です。結果に関係なく、どれだけの資本を投入したかを測定します。

### 勝率

利益で終了したクローズド取引の割合です。

```
勝率 = 勝ちトレード数 / (勝ちトレード数 + 負けトレード数) × 100
```

クローズドポジションのみが対象です。オープンポジションは勝率の計算から除外されます。

### 総取引数

選択した期間中にクローズしたポジションの数です。

### 平均P&L

クローズドトレードあたりの平均損益です。典型的なトレードが利益を出しているか、またどの程度かを示します。

### 平均保有時間

エグジット戦略または手動でクローズされるまでの平均ポジション保有時間です。時間と分で表示されます。

短い平均保有時間は通常、動きの速いミームコイン取引を示しています。長い保有時間は、より大型のトークンを取引しているか、より広いトレーリングストップを使用している可能性があります。

### ベストトレード

期間中で最も高い実現利益を出した単一のトレードです。上振れの可能性を理解し、戦略が十分にウィナーを伸ばしているかどうかを把握するのに役立ちます。

### ワーストトレード

期間中で最も深い実現損失を出した単一のトレードです。この数値が平均P&Lに比べて不釣り合いに大きい場合、ストップロスの引き締めが必要かもしれません。

## デイリーPnLチャート

90日間の日次純損益を示す棒グラフです。緑のバーは利益の出た日、赤のバーは損失の日を表します。任意のバーにホバーすると、その日の正確な数値と取引数が表示されます。

このチャートはパターンの発見に役立ちます: 一貫して利益を出しているか、それとも数回の大きな日が平均を引き上げているか？ 損失の日は時間とともに小さくなっているか？

## エグジット理由の内訳

選択した期間中にエグジットのトリガーとなったものの視覚的な内訳:

| 理由 | 意味 |
| :--- | :--- |
| **Target Profit** | テイクプロフィットの閾値に達した |
| **Trailing Stop** | 価格がピークからトレーリングパーセンテージ分下落した |
| **Stop Loss** | 価格がストップロスの閾値に達した |
| **Graduation** | トークンがボンディングカーブからDEXに移行した |
| **Deadline** | 設定したデッドライン後にポジションがタイムアウトした |

この内訳は、戦略を調整するための最も有用なツールの一つです。例えば:

- ほとんどのエグジットが**Stop Loss**の場合、ストップが厳しすぎるか、エントリーのタイミングが悪い可能性があります。
- ほとんどのエグジットが**Deadline**の場合、ポジションが横ばいになっている可能性があります。取引しているトークンに戦略に見合ったボラティリティがあるか検討してください。
- **Trailing Stop**が支配的でP&Lが健全な場合、戦略はうまく機能している可能性が高く、上昇を捉えてプルバックでエグジットしています。
- **Target Profit**が頻繁に発動するが、ベストトレードがターゲットよりはるかに高い場合、利益を取り損ねている可能性があります。ターゲットを拡大するか、トレーリングストップへの切り替えを検討してください。

## APIアクセス

アナリティクスはLaserSell APIを通じてプログラムで利用できます:

```
GET /v1/stats
```

上記のすべてのメトリクスを含む4つの期間すべてを返します。レスポンスフォーマットの詳細については[APIドキュメント](/api/exit-api/history)をご覧ください。
