LaserSellは、Solanaネットワーク向けの自動エグジットエンジンです。オンチェーンでウォレットを監視し、購入した新しいトークンを検出し、リアルタイムでPnLを追跡し、設定した条件が満たされると自動的に売却を実行します。鍵は常にあなたの管理下にあります。

## 仕組み

1. **あなたがトークンを購入します。** お使いの任意のプラットフォームで購入できます。
2. **LaserSellが購入を検出します。** オンチェーンであなたのウォレットを監視しています。
3. **エグジット戦略が自動的に実行されます。** LaserSellは、すべての価格更新においてテイクプロフィット、ストップロス、トレーリングストップ、デッドライン、卒業時売却の条件を評価します。
4. **条件が満たされた瞬間にLaserSellが売却します。** トランザクションが構築、署名され、Solanaネットワークに直接ブロードキャストされます。

手動でのクリック、確認ポップアップ、タブの切り替えは一切不要です。

## LaserSellではないもの

- **カストディアルサービスではありません。** 私たちがあなたの秘密鍵にアクセスすることは決してありません。
  - **デスクトップアプリ:** 鍵はあなたのマシン上でローカルに暗号化・保管されます。すべてのトランザクションに自分で署名します。
  - **Web App:** 鍵は[PrivyのTrusted Execution Environment（TEE）](https://docs.privy.io/guide/security)に保管されます。TEEはハードウェアで隔離されたエンクレーブであり、LaserSellもPrivyの担当者もアクセスできません。Privyは[Stripeの子会社](https://privy.io/blog/announcing-our-acquisition-by-stripe)です。鍵はいつでもエクスポートできます。
- **投資アドバイスではありません。** LaserSellは実行インフラです。あなたが戦略を設定し、LaserSellがそれを実行します。

## トレーディング方法を選ぶ

LaserSellはあなたのスタイルに合わせて利用できます。すべてのクライアントは同じ監視インフラストラクチャに接続し、同じ戦略を実行します。

### Web App

[LaserSell Web App](/web-app/overview)は、最も簡単に始められる方法です。サインインし、ウォレットに入金し、戦略を設定すれば、ブラウザを閉じていても24時間365日LaserSellがあなたの代わりに取引します。

- ダウンロード不要、インストール不要、あらゆるデバイスで動作
- PrivyのTrusted Execution Environmentによる自動実行
- ライブPnL更新付きのリアルタイムポジション監視
- 任意のウォレットの取引を自動ミラーリングするコピートレーディング
- 不在中の取引イベントのプッシュ通知

[Web Appを始める →](/web-app/overview)

### デスクトップアプリ

[LaserSellデスクトップアプリ](/desktop/overview)は、完全なローカルコントロールを求めるトレーダー向けに構築されています。鍵はあなたのマシンに保管され、すべてのトランザクションに自分で署名し、独自のRPCとトランザクション送信者を選択できます。

- リアルタイムポジション監視を備えたネイティブmacOSアプリケーション
- マルチレベルエグジット戦略とミラートレーディング
- スワップ機能付き内蔵Solanaウォレット
- デイリーPnLチャート、アチーブメント、インサイト付きアナリティクス
- Touch IDアンロックと7言語対応

[デスクトップアプリを始める →](/desktop/installation)

### APIとSDK

同じエグジットエンジンが、独自のクライアント、ボット、トレーディングプラットフォームを構築したい開発者向けに[LaserSell APIとSDK](/api/overview)を通じて利用可能です。

- [TypeScript](/api/sdk/typescript)、[Python](/api/sdk/python)、[Rust](/api/sdk/rust)、[Go](/api/sdk/go)で利用可能
- トランザクション構築APIとリアルタイムストリームへのフルアクセス
- カスタム戦略、アラート、ダッシュボード、自律エージェントの構築

[APIを探索する →](/api/overview)

## LaserSellのユーザー

### トレーダー

Solanaでミームコインを取引しているなら、LaserSellは自動エグジットを提供するため、すべてのポジションを手動で監視する必要がありません。戦略を一度設定すれば、あとは自動で実行されます。

- テイクプロフィット、ストップロス、トレーリングストップ、デッドラインタイムアウト、卒業時売却がすべて並行して実行されます。
- あらゆるエントリー方法に対応しています。お使いの任意のプラットフォームやウォレットから購入できます。

### デプロイヤー

RapidLaunch、Uxento、J7 Trackerなどのツールを使用してトークンを発行する場合、LaserSellはあなたの個人的なエグジットプランを自動化するため、ローンチとコミュニティに集中できます。

### 開発者

LaserSellのリアルタイムエグジットインテリジェンス、マルチプロトコル対応、トランザクション構築機能の上に、独自のトレーディングボット、プラットフォーム、カスタム戦略を構築できます。

## 対応プロトコル

LaserSellは、主要なSolanaのローンチパッドとDEXをカバーしています:

- Pump.funとPumpSwap
- Meteora DBCとDAMM V2
- Raydium LaunchpadとCPMM

詳細: [プラットフォームとコイン対応](/onboarding/platform-and-coin-support)
