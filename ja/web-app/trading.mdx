Tradingページは、Web Tradingのメインコントロールセンターです。**Positions**タブでアクティブな取引を監視し、**Copy Trading**タブで他のウォレットをミラーリングできます。

## はじめに

まだWeb Tradingセッションを開始していない場合、Tradingページにはセットアップフォームが表示されます。

1. 監視する**ウォレットを選択**（登録済みウォレットから）
2. **エグジット戦略を設定**（目標利益、ストップロス、トレーリングストップ、最大スリッページ）
3. **Start Web Trading**をクリック

開始すると、LaserSellがウォレットを監視し、戦略の閾値に達したときに自動的に取引を実行します。

## Positionsタブ

Positionsタブは3カラムレイアウトを使用しています。

- **左**: 損益サマリー付きの全オープンポジション一覧
- **中央**: 選択したポジションの詳細ビュー（トークン情報、エントリー価格、現在の利益、売却ボタン）
- **右**: エグジット戦略設定（いつでも編集可能）

### エグジット戦略の設定

| 設定 | 説明 |
|---------|-------------|
| **Target Profit %** | 利益がこのパーセンテージに達したら売却 |
| **Stop Loss %** | 損失がこのパーセンテージに達したら売却 |
| **Trailing Stop %** | 価格をこのパーセンテージ分トレーリングして利益を確保 |
| **Max Slippage %** | 自動売却時に許容する最大スリッページ（デフォルト5%） |
| **Sell on Graduation** | トークンがpump.funからRaydiumに卒業したときに自動売却 |

変更はすべてのアクティブポジションに即座に適用されます。

### 手動売却

任意のポジションの**Sell Now**をクリックすると、戦略の閾値に関係なく即座に売却を実行します。

## Copy Tradingタブ

Copy Tradingを使用すると、他のウォレットの取引を自動的にミラーリングできます。

### ウォッチウォレットの追加

1. **Add Watch Wallet**をクリック
2. Solanaウォレットアドレスまたは`.sol`ドメイン名（SNS経由で解決）を入力
3. ニックネームと自動購入金額（SOL単位）を設定
4. ウォッチリストにウォレットが追加される

ウォッチ対象のウォレットがトークンを購入すると、LaserSellがあなたのウォレットで同じ購入を自動的に実行します。

### ミラー設定

| 設定 | 説明 |
|---------|-------------|
| **Max positions per wallet** | ミラーポジションの同時保有数を制限 |
| **Cooldown** | ミラー購入間の最小間隔 |
| **Skip creator tokens** | ウォッチ対象ウォレットが作成したトークンを無視 |
| **Max active SOL** | ミラーポジション全体に投入するSOLの上限 |
| **Buy slippage** | ミラー購入時の最大スリッページ |
| **Min pool liquidity** | 流動性の低いトークンをスキップ |
| **Max entry drift** | ウォッチ対象ウォレットのエントリーから価格が離れすぎている場合はスキップ |
| **Max consecutive losses** | 連続N回の損失後にウォッチウォレットを自動無効化 |

## セッションコントロール

ヘッダーの**エージェントトグル**（電源アイコン）を使用してWeb Tradingを開始または停止します。ステータスドットの表示は以下の通りです。

- 🟢 **緑**: アクティブで監視中
- 🔵 **青**: 新しいポジションを待機中
- 🟡 **黄**: 一時停止中（エラーまたは非アクティブ）
- ⚪ **灰**: 停止中

<Warning>
  デスクトップアプリとWeb Appの両方で同じウォレットを監視している場合、閾値に達すると両方が売却を試みます。一方から他方に切り替える際は、使用しない方をオフにしてください。
</Warning>

## ティア制限

サブスクリプションティアによって、Web Tradingで利用可能な容量が決まります。これらの制限はデスクトップアプリとAPIにも同様に適用されます。

| | Free | Professional | Advanced |
|--|------|-------------|----------|
| **ウォレット** | 1 | 5 | 200 |
| **ウォレットごとのポジション** | 2 | 100 | 100 |
| **ウォッチウォレット** | 2 | 10 | 25 |
| **セッションタイムアウト** | 45分 | なし | なし |
| **常時監視** | 不可 | 可能 | 可能 |
| **Exit Ladder** | 不可 | 可能 | 可能 |
| **Liquidity Guard** | 不可 | 可能 | 可能 |
| **Behavioral Insights** | 不可 | 可能 | 可能 |

Freeティアのユーザーは基本的なエグジット戦略（目標利益、ストップロス、トレーリングストップ）でWeb Tradingを全機能利用できます。Professionalにアップグレードすると、高度な戦略、常時監視、より大きな容量が利用可能になります。
