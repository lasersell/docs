LaserSell Web Appは、ブラウザに自動トレーディングを提供します。エグジット戦略を設定し、ウォレットに資金を入金すると、LaserSellがブラウザを閉じている間も24時間365日あなたの代わりに取引を実行します。

## 仕組み

<Steps>
  <Step title="ウォレットを設定する">
    サインインすると、[Privy](https://privy.io)によって安全なウォレットが自動的に作成されます。既存のウォレットをインポートすることも可能です。秘密鍵はTrusted Execution Environment（TEE）に保管され、LaserSellサーバーに触れることはありません。
  </Step>
  <Step title="ウォレットに入金する">
    外部ウォレット（Phantom、Backpackなど）からウォレットアドレスにSOLを送信します。ウォレットアドレスと残高はWalletページで確認できます。
  </Step>
  <Step title="戦略を設定する">
    エグジット戦略を設定します。目標利益、ストップロス、トレーリングストップ、最大スリッページを指定します。これらのルールに基づいて、LaserSellがポジションを自動的に売却するタイミングを決定します。
  </Step>
  <Step title="取引を開始する">
    「Start Web Trading」をクリックすると、LaserSellがウォレットの監視を開始します。ポジションが戦略の閾値に達すると、ウォレットを使用して自動的に取引が実行されます。
  </Step>
</Steps>

## Web App とデスクトップアプリの比較

両方のクライアントは同じLaserSellインフラストラクチャに接続しますが、内部の動作は異なります。

| 機能 | デスクトップアプリ | Web App |
|---------|------------|---------|
| **署名** | ローカルのキーペア | Privy TEEウォレット（委任署名） |
| **オフライン実行** | 不可（アプリを開いている必要あり） | 可能（不在中もサーバーが実行） |
| **ウォレット** | ローカルでインポート/生成 | Privy埋め込みまたはインポート |
| **コピートレーディング** | 対応 | 対応 |
| **ポジション監視** | 対応 | 対応 |
| **コインの購入** | 対応 | 未対応 |
| **トークン検索** | 対応 | 未対応 |

## 主要コンセプト

### Privy ウォレット

Web Tradingウォレットは[Privy](https://privy.io)によって管理されています。Privyは[Stripeに買収された](https://privy.io/blog/announcing-our-acquisition-by-stripe)ウォレットインフラ企業です。Privyはあなたの秘密鍵を**Trusted Execution Environment**（TEE）に保管します。TEEはハードウェアで隔離されたエンクレーブであり、LaserSellを含め誰もあなたの鍵に直接アクセスすることはできません。

取引の実行が必要な場合、LaserSellはPrivyにトランザクションの署名をリクエストします。PrivyはTEE内で署名し、署名済みトランザクションを返します。その後、LaserSellがSolanaネットワークに送信します。

### 常時監視

Web Tradingがアクティブな間、LaserSellはあなたの代わりに監視インフラストラクチャへの永続的な接続を維持します。ポジションはリアルタイムで追跡され、戦略の閾値に達するとエグジットシグナルが即座に実行されます。

ダッシュボードヘッダーの電源アイコンを使用してWeb Tradingのオン/オフを切り替えることができます。デスクトップアプリに切り替える際は、同じウォレットに対する重複した売却を避けるためにWeb Tradingをオフにしてください。

### セッション状態

| 状態 | 意味 |
|-------|--------------|
| **Active** | ストリーム接続中、ポジション監視中、取引が自動実行される |
| **Paused** | 一時停止中（エラーまたは手動停止） |
| **Stopped** | Web Tradingがオフ、リソース消費なし |
