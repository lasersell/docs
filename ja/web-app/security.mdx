Web Appはデスクトップアプリとは異なるセキュリティモデルを使用しています。それぞれの違いを理解することで、どちらのクライアントを使用するかを十分な情報に基づいて判断できます。

## 信頼モデル

| | デスクトップアプリ | Web App |
|--|------------|---------|
| **モデル** | ノンカストディアル（ローカル署名） | ノンカストディアル（Privy TEEによる委任署名） |
| **署名者** | あなた自身（ローカル） | PrivyのTEE（あなたの代わりに署名） |
| **送信者** | あなたが選択（RPC、Heliusなど） | LaserSellのサーバー（自社RPC経由） |
| **鍵の保管場所** | あなたのマシン（暗号化） | PrivyのTrusted Execution Environment |
| **LaserSellが鍵を見るか？** | 見ない | 見ない（TEEはハードウェアで隔離） |

## Privy TEE

[Privy](https://privy.io)は[Stripeに買収された](https://privy.io/blog/announcing-our-acquisition-by-stripe)ウォレットインフラ企業です。Privyはあなたの秘密鍵を**Trusted Execution Environment**（TEE）に保管します。TEEはハードウェアで隔離されたエンクレーブです。鍵はトランザクションの署名時でもTEEから外に出ることはありません。LaserSellがPrivyにトランザクションの署名をリクエストし、PrivyがTEE内で署名し、署名済みの結果のみを返します。

これが意味すること:
- LaserSellサーバーがあなたの秘密鍵にアクセスすることは決してない
- LaserSellサーバーが侵害されても鍵が漏洩することはない
- PrivyのセキュアUIからいつでも鍵をエクスポートできる

## ウォレット所有権の検証

ウォレットを登録する際、Web AppはPrivyの`signMessage` APIを使用して証明メッセージ（`lasersell-register:<pubkey>:<timestamp>`）に署名します。このed25519署名がLaserSellのサーバーによって検証され、あなたがウォレットを管理していることを証明します。これはデスクトップアプリで使用されるものと同じ検証フローです。

## トランザクションフロー

自動売却がトリガーされた場合:

1. LaserSellの監視サーバーが戦略の閾値到達を検出
2. 未署名のトランザクションが構築される（デスクトップと同じ）
3. LaserSellがPrivyにトランザクションの**署名**をリクエスト（署名のみ、送信はしない）
4. PrivyがTEE内で署名し、署名済みトランザクションを返す
5. LaserSellが署名済みトランザクションを自社RPCを通じてSolanaに送信
6. トランザクション署名を含む通知を受信

この一連のプロセスにおいて、LaserSellのサーバーがあなたの秘密鍵を見ることは一切ありません。署名と鍵の保管は完全にPrivyのインフラストラクチャ内で行われます。

## データ保護

| データ | 保管場所 | アクセス |
|------|---------|--------|
| APIキー | Supabaseで暗号化 | カラムレベルのセキュリティにより、ブラウザから生のキーへのアクセスを防止 |
| ウォレットの秘密鍵 | Privy TEE | LaserSellからアクセス不可 |
| トレーディングセッション状態 | Supabase（RLS保護） | 自分のセッションのみ表示 |
| 通知 | Supabase（RLS保護） | 自分の通知のみ表示 |
| プッシュサブスクリプション | Supabase（RLS保護） | ユーザー管理 |

## Privyから鍵を削除する

Privyのインフラストラクチャから秘密鍵を完全に削除する場合:

### インポートしたウォレット
1. ダッシュボードの**Wallet**ページに移動
2. インポートしたウォレットカードのゴミ箱アイコンをクリック
3. LaserSellからウォレットが削除され、PrivyのTEEから鍵も削除される
4. エンクレーブ内で鍵が完全に破棄される

### 埋め込みウォレット
埋め込みウォレットはPrivyアカウントに紐付けられています。削除するには:
1. まず**秘密鍵をエクスポート**（ウォレットカードの「Export Key」をクリック）
2. エクスポートした鍵を自分のウォレット（Phantom、Backpackなど）に安全に保管
3. [サポート](https://docs.lasersell.io/getting-support/support)に連絡してPrivyアカウントの削除をリクエスト
4. Privyアカウントが削除されると、埋め込みウォレットの鍵は完全に破棄される

<Warning>
  ウォレットまたはPrivyアカウントの削除は元に戻せません。秘密鍵をエクスポートしていない場合、そのウォレットの資金へのアクセスを永久に失います。
</Warning>

## 推奨事項

- Web Tradingには、メインの保有用ウォレットではなく**専用のトレーディングウォレット**を使用してください
- デスクトップアプリに切り替える際は、重複した売却を避けるために**Web Tradingをオフ**にしてください
- ウォレットをインポートした後は、バックアップとして**秘密鍵をエクスポート**してください
- 不在中の実行時にMEVから保護するため、エグジット戦略の**最大スリッページ**設定（デフォルト5%）を確認してください
