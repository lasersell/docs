---
title: "GET /v1/account"
description: "Retrieve your account tier, plan name, and usage limits from the LaserSell API."
---

## Endpoint

```
GET https://api.lasersell.io/v1/account
```

## Headers

| Header      | Required | Description            |
|-------------|----------|------------------------|
| `x-api-key` | Yes      | Your LaserSell API key |

## Response: `AccountResponse`

| Field       | Type     | Description                                      |
|-------------|----------|--------------------------------------------------|
| `tier`      | `number` | Numeric tier identifier (`0`, `1`, or `2`).      |
| `tier_name` | `string` | Human readable plan name (`free`, `tier_1`, `tier_2`). |
| `limits`    | `object` | Usage limits for your plan (see below).          |

### `limits` Object

| Field                       | Type     | Description                                           |
|-----------------------------|----------|-------------------------------------------------------|
| `max_wallets`               | `number` | Maximum wallets you can track simultaneously.         |
| `max_positions_per_wallet`  | `number` | Maximum open positions per wallet.                    |
| `max_positions_per_session` | `number` | Maximum total open positions across all wallets.      |

## curl Example

```bash
curl https://api.lasersell.io/v1/account \
  -H "x-api-key: YOUR_API_KEY"
```

## Example Response

```json
{
  "tier": 1,
  "tier_name": "tier_1",
  "limits": {
    "max_wallets": 5,
    "max_positions_per_wallet": 100,
    "max_positions_per_session": 500
  }
}
```
