## 安装

```bash
cargo add lasersell-sdk
cargo add tokio --features full
```

关于 API 客户端用法、退出智能流会话、流动性快照、交易辅助函数和错误处理的完整代码示例，请参阅[英文版](/api/sdk/rust)。所有 API 方法名和签名与英文版完全一致。
