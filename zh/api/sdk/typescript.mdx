## 安装

```bash
npm install @lasersell/lasersell-sdk
```

包附带 TypeScript 声明，支持 ESM 和 CommonJS。

关于 API 客户端用法、退出智能流会话、通道分离、流动性快照、交易辅助函数和错误处理的完整代码示例，请参阅[英文版](/api/sdk/typescript)。所有 API 方法名和签名与英文版完全一致。
