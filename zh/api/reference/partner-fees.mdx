## 概述

LaserSell API 支持在任何买入或卖出交易上添加合作伙伴费用。这让集成方可以收取路由到指定钱包的费用，作为兑换指令的一部分。

合作伙伴费用是可选的，按每个请求配置。它们与 LaserSell 自己的协议费用**无关**。

## 费用模式

你可以用两种方式之一指定费用。这两种模式互斥；不要在同一请求上同时设置。

### 基点模式

设置 `partner_fee_bps` 以收取兑换价值的百分比。

| 字段                   | 类型     | 说明                                |
|-------------------------|----------|--------------------------------------------|
| `partner_fee_recipient` | `string` | 接收方钱包地址（base58 公钥）。  |
| `partner_fee_bps`       | `number` | 费用（基点）。最大 `50`（0.5%）。      |

### 固定 Lamports 模式

设置 `partner_fee_lamports` 以收取固定金额。

| 字段                   | 类型     | 说明                                |
|-------------------------|----------|--------------------------------------------|
| `partner_fee_recipient` | `string` | 接收方钱包地址（base58 公钥）。  |
| `partner_fee_lamports`  | `number` | 费用（lamports）。最大 `50,000,000`（0.05 SOL）。 |

## 限制

| 模式       | 最大值            | 等值         |
|------------|--------------------------|--------------------|
| bps        | 50 bps                   | 0.5%               |
| lamports   | 50,000,000 lamports      | 0.05 SOL           |

超过这些限制的请求将被拒绝并返回 `400` 错误。

关于每种 SDK 语言中使用基点费用和固定 lamports 费用的完整代码示例，请参阅[英文版](/api/reference/partner-fees)。

## 企业：自定义变现

处理大量交易的 Enterprise 合作伙伴可以获得超越标准合作伙伴费用系统的自定义变现方案。

如需讨论自定义方案，请联系 [sales@lasersell.io](mailto:sales@lasersell.io)。

## 重要说明

- 合作伙伴费用在 **Advanced** 和 **Enterprise** 等级可用。参见[速率限制和等级](/api/reference/rate-limits)。
- `partner_fee_recipient` 钱包必须是有效的 base58 Solana 公钥。
- 接收方不需要已有账户；费用指令会在必要时创建一个。
- 在同一请求上同时设置 `partner_fee_bps` 和 `partner_fee_lamports` 将导致 `400` 错误。
- 合作伙伴费用从兑换输出中扣除，在同一交易中原子化路由。
