使用 LaserSell 平台相关费用的透明说明。

LaserSell 采用**分级路由费**，直接在链上执行，以实现最大速度和透明度。你的费率取决于你的订阅等级。

## 1. 路由费

| 等级 | 费率 | 最低费用 |
|------|----------|-------------|
| Personal（免费） | 2%（200 基点） | 每笔交易 0.01 SOL |
| Tier 1 — Professional（$99/月） | 0.9%（90 基点） | 每笔交易 0.01 SOL |
| Tier 2 — Advanced（$249/月） | 0.5%（50 基点） | 无 |
| Enterprise | 自定义 | 自定义 |

- **简单且可预测：** 没有复杂的利润计算，没有隐藏的价差。你按上述百分比支付输出金额的费用，受你所在等级的最低费用约束。
- **链上执行：** 费用由 LaserSell Router 链上程序原子化处理。在交易过程中自动从 SOL 或 USD1 收益中扣除。USD1 交易仅使用百分比费用（不适用 SOL 最低费用）。

## 2. 网络和基础设施费用

为确保你的交易即时落地，涉及两项标准网络费用。这些是直接支付给 Solana 网络的过手费用，不是支付给我们的：

- **Jito 小费（贿赂）：** 直接支付给验证者，以便你的交易在同一区块中优先于其他交易。
- **Solana 优先费：** 在网络上处理高优先级计算所需的标准区块链费用。
