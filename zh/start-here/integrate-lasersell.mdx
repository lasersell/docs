如果你的产品涉及 Solana 代币交易，LaserSell 可以为其提供执行层支持。

驱动 LaserSell 客户端的退出智能引擎同样通过 [LaserSell SDK 和 API](/api/overview) 提供。Telegram 机器人、Web 平台、移动应用、浏览器扩展、CLI 工具、投资组合管理器、发射台服务、跟单交易系统、信号群组、分析仪表板。如果你的用户在 Solana 上买卖代币，LaserSell 为你提供基础设施，通过实时策略评估、多协议交易构建和内置变现功能实现自动化。

你不需要构建退出引擎。不需要追踪 AMM 状态或处理协议特定的指令。只需集成一次，即可获得 Solana 上所有主要发射台和 DEX 的覆盖。

## 实时退出智能

退出智能流是一个持久的 WebSocket 连接，它监控钱包、追踪仓位、实时评估策略，并在阈值满足时提供预构建的退出交易。

无需从零开始构建自己的盈亏追踪、价格监控和退出逻辑，你只需连接到流并接收预构建的交易。你的应用定义策略（止盈、止损、追踪止损、截止时间、毕业即卖），LaserSell 处理其余一切。

这意味着你可以为用户提供自动退出功能，而无需自己构建或维护执行引擎。

## 买入和卖出交易构建器

REST API 通过 `/v1/buy` 和 `/v1/sell` 按需构建未签名的兑换交易。完整协议覆盖 Pump.fun、PumpSwap、Meteora DBC、Meteora DAMM V2、Raydium Launchpad 和 Raydium CPMM。

你不需要追踪 AMM 状态、计算路由或处理协议特定的指令格式。只需发送包含代币地址和金额的请求，API 就会返回一个优化的未签名交易，随时可以签名并提交。

## 你的密钥，你来签名

API 返回的每笔交易都是未签名的。你可以按照你的架构需要来处理签名。没有任何密钥会接触 LaserSell 基础设施。

无论你的应用是代表用户管理钱包，还是让用户直接签名，LaserSell 都能适配你现有的密钥管理方案，不会对你的签名方式施加任何限制。

## 内置变现

你的用户通过 LaserSell 执行的每笔交易都可以包含直接路由到你钱包的合作伙伴费用。你在每个请求上设置费率（基点或固定 lamports），它作为兑换的一部分在链上原子化执行。

无需开票、无需对账、无需链下支付流程。你的收入在交易发生的同一笔交易中到达你的钱包。

请参阅[合作伙伴费用](/api/reference/partner-fees)了解实现细节。企业合作伙伴可以与我们协商自定义变现方案。

## 多协议覆盖

单次集成即可让你的产品覆盖 Solana 上的主要交易协议：

- **Pump.fun** 联合曲线和 **PumpSwap** 市场
- **Meteora** 动态联合曲线（DBC）和动态 AMM V2
- **Raydium** Launchpad 和 CPMM 池

随着 Solana 上新协议的推出，LaserSell 会添加支持，你的集成会自动获取。无需在你这边做任何代码更改。

## LaserSell 的适用场景

任何用户持有 Solana 代币并需要卖出的产品都可以受益于 LaserSell。以下是一些示例，但这并非详尽列表。如果你的产品涉及 Solana 上的代币交易，退出智能很可能在你的技术栈中有一席之地。

### 机器人（Telegram、Discord、CLI 或自定义）

为你的机器人用户在每笔交易上提供自动止盈、止损和追踪止损。按用户钱包连接退出智能流，在条件满足时发送退出通知或自动执行卖出。合作伙伴费用让你从机器人处理的每笔交易中获利。

### Web 和移动交易平台

为你的交易界面添加一键买卖和自动退出策略。交易构建器处理协议路由，退出智能流提供实时盈亏更新以驱动实时仪表板。

### 发射台和发行工具

为代币创建者自动化发行后的退出工作流。当发行者通过你的工具发行时，LaserSell 可以监控仓位并自动执行创建者的退出计划。

### 跟单交易和信号群组

当信号触发或领导者建仓时，LaserSell 可以自动为每个跟随者管理退出。每个钱包都有自己的策略评估和退出执行，无需任何人工干预。

### 投资组合追踪器、分析和告警

连接退出智能流，获取所有支持协议的实时仓位追踪、盈亏更新和退出事件数据。驱动分析仪表板、告警系统或风险管理工具。

### 浏览器扩展和钱包插件

将自动退出功能直接嵌入浏览体验。用户在任何平台上买入，你的扩展使用 LaserSell 在后台管理卖出。

### 其他任何场景

如果你的用户在 Solana 上交易代币，LaserSell 的退出引擎和交易构建器可以直接集成。从你的角度来看，API 是协议无关的。你发送代币地址和金额，就会得到一个优化的交易。退出智能流监控钱包并实时发送退出信号。如何向用户呈现这些完全取决于你。
