LaserSell 是 Solana 网络上的自动退出引擎。它监控你的链上钱包，检测你买入的新代币，实时跟踪你的盈亏，并在满足你设定的条件时自动执行卖出。你的密钥始终在你的掌控之下。

## 工作原理

1. **你买入一个代币**，可以在你已经使用的任何平台上操作。
2. **LaserSell 检测到买入**，通过监控你的链上钱包实现。
3. **你的退出策略自动运行。** LaserSell 在每次价格更新时评估止盈、止损、追踪止损、截止时间和毕业即卖条件。
4. **当任一条件满足时，LaserSell 立即卖出。** 交易被构建、签名并直接广播到 Solana 网络。

无需手动点击，无需确认弹窗，无需在标签页之间切换。

## LaserSell 不是什么

- **不是托管服务。** 我们永远无法访问你的私钥。
  - **桌面应用：** 密钥在你的设备上加密存储，永不传输。你签署每一笔交易。
  - **Web App：** 密钥存储在 [Privy 的可信执行环境（TEE）](https://docs.privy.io/guide/security)中，这是一个硬件隔离的安全区域，LaserSell 和 Privy 的工作人员都无法访问。Privy 是一家 [Stripe 旗下公司](https://privy.io/blog/announcing-our-acquisition-by-stripe)。你可以随时导出你的密钥。
- **不是投资建议。** LaserSell 是执行基础设施。你设定策略，LaserSell 执行它。

## 选择你的交易方式

LaserSell 随时为你服务。每个客户端都连接到相同的监控基础设施，执行相同的策略。

### Web App

[LaserSell Web App](/web-app/overview) 是最简单的入门方式。登录，为钱包充值，设定策略，LaserSell 就会全天候为你交易，即使浏览器已关闭也不影响。

- 无需下载，无需安装，任何设备都可以使用
- 通过 Privy 的可信执行环境实现自动执行
- 实时仓位监控及盈亏更新
- 跟单交易，自动镜像任何钱包的操作
- 离开时通过推送通知接收交易事件

[开始使用 Web App →](/web-app/overview)

### 桌面应用

[LaserSell 桌面应用](/desktop/overview)专为需要完全本地控制的交易者打造。你的密钥保存在你的设备上，你签署每一笔交易，你可以选择自己的 RPC 和交易发送器。

- 原生 macOS 应用，实时仓位监控
- 多级退出策略和镜像交易
- 内置 Solana 钱包，支持兑换
- 数据分析，包含每日盈亏图表、成就和洞察
- Touch ID 解锁，支持 7 种语言

[开始使用桌面应用 →](/desktop/installation)

### API 和 SDK

同样的退出引擎也通过 [LaserSell API 和 SDK](/api/overview) 提供给希望构建自己客户端、机器人或交易平台的开发者。

- 提供 [TypeScript](/api/sdk/typescript)、[Python](/api/sdk/python)、[Rust](/api/sdk/rust) 和 [Go](/api/sdk/go) 版本
- 完整访问交易构建 API 和实时流
- 构建自定义策略、警报、仪表板或自主代理

[探索 API →](/api/overview)

## 谁在使用 LaserSell

### 交易者

如果你在 Solana 上交易 meme 币，LaserSell 为你提供自动退出功能，让你无需手动监控每个仓位。只需设置一次策略，然后让它自动运行。

- 止盈、止损、追踪止损、截止时间和毕业即卖功能并行运行。
- 兼容任何入场方式。你可以在任何你已经使用的平台或钱包上买入。

### 发行者

如果你使用 RapidLaunch、Uxento 或 J7 Tracker 等工具发行代币，LaserSell 可以自动化你的个人退出计划，让你专注于发行和社区建设。

### 开发者

在 LaserSell 的实时退出智能、多协议支持和交易构建能力之上，构建你自己的交易机器人、平台或自定义策略。

## 支持的协议

LaserSell 覆盖 Solana 上的主要发射台和 DEX：

- Pump.fun 和 PumpSwap
- Meteora DBC 和 DAMM V2
- Raydium Launchpad 和 CPMM

详情请参阅：[平台和代币支持](/onboarding/platform-and-coin-support)
