编程技能为你的 AI 编程助手提供 LaserSell SDK 的深度嵌入知识。助手无需按需搜索文档，而是将 SDK 模式、方法签名、工作示例、策略配置和故障排除指南直接加载到其上下文窗口中。这意味着它可以从一开始就编写正确的 LaserSell 代码，无需查找任何内容。

## 技能与 MCP 的对比

LaserSell 提供两种 AI 集成路径。它们服务于不同目的，一起使用效果最佳。

| | MCP 服务器 | 编程技能 |
|---|---|---|
| **工作方式** | 助手需要查找时按需搜索 LaserSell 文档 | 在助手开始编写代码前直接将深度 SDK 知识加载到上下文中 |
| **最适合** | 查找 API 模式、找到特定文档页面、回答关于 LaserSell 的问题 | 编写 SDK 集成、构建交易、连接流、处理错误 |
| **知识何时可用** | 按需（助手必须主动请求） | 即时（助手上下文中已有） |
| **设置** | 将 MCP 服务器配置添加到编辑器 | 安装技能包 |

MCP 服务器适合发现和参考查找。编程技能适合编写与 SDK 集成的代码。两者结合使用可获得最佳体验。

## 包含内容

LaserSell 编程技能提供一个结构化知识树，助手在处理 LaserSell 代码时会加载：

- **SKILL.md** — 根技能文件，包含 SDK 概述、关键概念和详细参考的指引
- **4 种语言的工作示例** — TypeScript、Python、Rust 和 Go 示例，涵盖买卖交易、流连接、策略配置和交易提交
- **策略指南** — 目标利润、止损、追踪止损、毕业即卖和截止时间配置的详细参考
- **故障排除指南** — 常见错误、原因和修复方法
- **API 和类型参考** — 每个端点的请求/响应模式、枚举值和字段描述
- **流协议参考** — 完整的退出智能流消息格式、事件类型和连接生命周期
- **模板** — 常见集成模式的起始代码模板

## 安装

### 方式 1：npx（推荐）

```bash
npx skills add lasersell/skills
```

这会下载并安装 LaserSell 编程技能到你的项目中。助手在下次会话中自动识别。

### 方式 2：Claude Code CLI

```bash
claude install-skill https://github.com/lasersell/skills/tree/main/skills/lasersell
```

### 方式 3：手动

从 [LaserSell skills 仓库](https://github.com/lasersell/skills)复制 `skills/lasersell/` 目录到你项目的 `.claude/skills/` 文件夹：

```bash
mkdir -p .claude/skills
cp -r path/to/skills/lasersell .claude/skills/
```

助手会检测技能文件并在你处理 LaserSell 代码时将其加载到上下文中。

## 技能覆盖的内容

安装后，你的 AI 助手将获得以下主题的完整上下文：

**构建买入和卖出交易。** 助手知道如何在所有四种 SDK 语言中构建 `BuildBuyTxRequest` 和 `BuildSellTxRequest` 对象、调用 LaserSell API 和处理响应。它理解输出类型（SOL 与 USD1）、滑点配置和部分卖出。

**退出智能流。** 助手可以编写流连接代码、配置策略参数、处理所有服务器事件类型（`pnl_update`、`exit_signal_with_tx`、`position_opened`、`position_closed`、`liquidity_snapshot`），并端到端处理退出信号。

**部分和多段卖出。** 助手理解如何查询滑点带、确定给定滑点阈值下的最大可卖金额，以及构建部分卖出交易。它还可以为大仓位构建多段卖出序列。

**流动性快照。** 助手知道如何读取滑点带、解读流动性趋势（`growing`、`stable`、`draining`），以及使用快照数据优化卖出大小。

**交易签名和提交。** 助手可以编写本地交易签名代码，并理解所有发送目标：默认 RPC、自定义 RPC 和 Helius Sender。它知道何时使用每个以及如何配置。

**错误处理。** 助手具有常见错误代码、重试模式、滑点失败以及 REST API 和流的连接恢复的上下文。

## 结合技能和 MCP

为获得最佳 AI 编程体验，同时使用编程技能和 MCP 服务器：

1. **安装编程技能**，使你的助手在编写 LaserSell 代码时将深度 SDK 知识加载到上下文中。
2. **连接 MCP 服务器**，使你的助手在需要查找技能范围之外的内容（如合作伙伴费用详情、速率限制等级或更新日志条目）时可以搜索完整文档。

两者都激活后，助手可以立即编写正确的 SDK 代码（来自技能）并按需查找补充信息（来自 MCP）。

参见 [MCP 服务器](/ai-agents/mcp-server)了解 MCP 设置说明。
