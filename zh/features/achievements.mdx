LaserSell 追踪你的交易里程碑，并在你达成时授予成就。共有超过 85 个成就，分布在 10 个类别中，从任何人都能达成的普通里程碑到需要卓越表现的传说成就。

<Note>
解锁的成就将可兑换前所未有的奖励。LaserSell 没有代币，也不会有空投。
</Note>

## 成就如何运作

成就在每小时的循环中自动检查。只有通过 LaserSell 协议路由的交易才会计入成就进度。当你解锁新成就时，应用会发送通知。

隐藏成就在你解锁之前不会显示其条件。

## 稀有度等级

| 稀有度 | 说明 |
| :--- | :--- |
| **普通** | 入门级里程碑。你在开始交易时会自然解锁这些。 |
| **不常见** | 需要持续交易活动的中等成就。 |
| **稀有** | 需要真正投入或技巧的重要成就。 |
| **史诗** | 只有少数交易者才能达到的卓越里程碑。 |
| **传说** | 系统中最稀有的成就。需要非凡的交易量、一致性或单次卓越表现。 |

## 类别

### 交易量

累计交易量里程碑。你通过 LaserSell 交易越多，攀升越高。

### 交易笔数

总完成交易里程碑，从你的第一笔交易一直到数千笔。

### 连续天数

连续交易日里程碑。当你在一天内至少完成一笔通过 LaserSell 路由的交易时，该天计为连续交易日。缺席一天会重置你的当前连续记录。

你的当前连续天数和最长连续天数显示在应用的成就标签页顶部。

### 利润

以 SOL 计的累计已实现利润里程碑。

### 速度

在短时间内完成的盈利交易成就。奖励快速执行和快速退出。

### 单笔交易

基于单个仓位表现的成就，从稳健回报到非凡倍数。

### 胜率

有最低交易要求的胜率里程碑，以防止刷数据。在数百笔交易中保持高胜率是最困难的事情之一。

### 镜像

[镜像交易](/desktop/mirror-trading)成就。追踪你在监控钱包中的跟单交易活动和表现。

### 秘密

条件在你解锁之前不会被揭示的隐藏成就。你在交易过程中会自然发现这些。

### 精通

当你完成某个类别中的所有成就时授予的元成就。每个其他类别都有一个精通成就。完成所有秘密成就将获得唯一的传说级精通。

## 连续天数

你的交易连续天数追踪连续在 LaserSell 协议上完成至少一笔交易的天数。

- **当前连续天数** — 包括今天在内，你已连续交易了多少天。
- **最长连续天数** — 你的历史最高纪录。
- **最近交易日期** — 你最近一次交易被记录的时间。

缺席一天会将你的当前连续天数重置为零。你的最长连续天数将永久保留。

## API 访问

成就可通过编程方式访问：

```
GET /v1/achievements
```

返回所有成就及其解锁状态和时间戳。秘密类别中的已锁定成就返回遮蔽的描述。
