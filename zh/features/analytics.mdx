LaserSell 追踪你通过协议执行的每笔交易，并将其转化为可操作的表现数据。分析仪表板可在[桌面应用](/desktop/analytics)和 API 中使用。

## 时间周期

所有指标可按四个时间周期查看：

- **24 小时** — 今日表现
- **7 天** — 每周快照
- **30 天** — 每月概览
- **全部时间** — 你的完整交易历史

每个周期独立计算自己的指标集，因此你可以比较你的交易如何随时间演变。

## 关键指标

### 总盈亏

你在所选周期内所有已平仓位的净已实现利润或亏损。这是每笔交易最终利润的总和，以你选择的显示货币展示。

### 总交易量

该周期内所有入场金额（你购买代币时花费的 SOL 或 USD1）的总和。这衡量你投入了多少资金，与结果无关。

### 胜率

以盈利结束的已平仓交易的百分比。

```
胜率 = 盈利交易 / (盈利交易 + 亏损交易) × 100
```

只有已平仓的仓位才会计入胜率计算。未平仓的仓位不包含在内。

### 总交易笔数

在所选周期内平仓的仓位数量。

### 平均盈亏

每笔已平仓交易的平均利润或亏损。这告诉你你的典型交易是否盈利以及盈利多少。

### 平均持仓时间

你在仓位被平仓之前平均持有多长时间，无论是通过退出策略还是手动平仓。以小时和分钟显示。

较短的平均持仓时间通常表明快速的 meme 币交易。较长的持仓时间可能表明你在交易较大市值的代币或使用较宽的追踪止损。

### 最佳交易

该周期内已实现利润最高的单笔交易。有助于了解你的上行潜力以及你的策略是否让赢家充分发挥。

### 最差交易

该周期内已实现亏损最深的单笔交易。如果这个数字与你的平均盈亏相比不成比例地大，你的止损可能需要收紧。

## 每日盈亏图表

一个 90 天的柱状图，显示你每天的净利润或亏损。绿色柱表示盈利日，红色柱表示亏损日。将鼠标悬停在任意柱上可查看当天的确切数字和交易数量。

这个图表帮助你发现模式：你是持续盈利的，还是少数几个大幅盈利日拉高了平均值？你的亏损日是否在随时间变小？

## 退出原因分布

所选周期内触发退出原因的可视化分布：

| 原因 | 含义 |
| :--- | :--- |
| **目标利润** | 达到了你的止盈阈值 |
| **追踪止损** | 价格从峰值回落了你设定的追踪百分比 |
| **止损** | 价格达到了你的止损阈值 |
| **毕业** | 代币从联合曲线迁移到了 DEX |
| **截止时间** | 仓位在你配置的截止时间后超时 |

这个分布是调整策略最有用的工具之一。例如：

- 如果大多数退出是**止损**，你的止损可能设置得太紧，或者你可能在不好的时机入场。
- 如果大多数退出是**截止时间**，你的仓位可能在横盘。考虑你交易的代币是否有足够的波动性来配合你的策略。
- 如果**追踪止损**占主导且你的盈亏是健康的，你的策略很可能运作良好，在回调时捕获了上涨并退出。
- 如果**目标利润**频繁触发但你的最佳交易远高于你的目标，你可能在错过更多利润。考虑扩大你的目标或切换到追踪止损。

## API 访问

分析功能可通过 LaserSell API 以编程方式访问：

```
GET /v1/stats
```

返回所有四个时间周期及上述所有指标。请参阅 [API 文档](/api/exit-api/history)了解响应格式详情。
