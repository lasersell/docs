每个交易者都有盲点。洞察功能帮你找到你的盲点。

LaserSell 分析你的交易历史，生成专门为你量身定制的洞察。这些不是通用提示或一刀切的建议。它们是从你的实际交易、你的行为模式和你的策略配置中得出的具体观察。

洞察功能面向 Professional 及以上等级用户开放。

## 洞察如何运作

LaserSell 研究你在可配置周期（24 小时、7 天、30 天、90 天或全部时间）内的已平仓交易，寻找可能在帮助或损害你表现的模式。当发现值得呈现的内容时，你会收到一个带有严重程度级别和修改建议的洞察。

洞察会随着新交易的完成而更新。它们不是静态的。它们会随着你的交易一起演变。

## 严重程度级别

每个洞察都标记了一个严重程度，表示它对你表现的影响程度：

- **高** — 对你的盈亏有重大影响。优先处理这些。
- **中** — 可以改善的明显模式。
- **低** — 次要观察或积极趋势。

## 预期内容

洞察覆盖你交易行为的广泛范围。它们可能告诉你，你没有注意到的某个习惯正在让你亏钱；你策略中的某个设置没有根据你实际的交易方式进行校准；你在某些时段或某些市场上表现更好；或者你的整体轨迹正朝着错误的方向发展。

有些洞察是鼓励性的。大多数是建设性的。所有洞察都基于你的数据实际展示的内容，而不是关于你应该如何交易的假设。

## API 访问

洞察可通过编程方式访问：

```
GET /v1/insights?period=7d
```

支持的周期：`24h`、`7d`、`30d`、`90d`、`all`。需要 Professional 及以上等级。免费等级返回空响应，带有 `capped: true`。
