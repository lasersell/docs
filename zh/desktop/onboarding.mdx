## 步骤 1：登录

使用你的 LaserSell 账户登录以获取 API 密钥。你有两个选项：

- **浏览器登录：** 点击"Sign In"并在浏览器中进行认证。应用通过深度链接自动接收你的 API 密钥。
- **手动粘贴：** 如果你已有 API 密钥，直接粘贴即可。

如果你还没有账户，点击"Sign Up"创建一个。账户免费且即时创建。

## 步骤 2：钱包设置

你需要至少一个 Solana 钱包供 LaserSell 监控。选择一种方式：

- **导入现有钱包。** 粘贴你的 Base58 私钥。如果你已有一个有资金的交易钱包，这很有用。
- **生成新钱包。** LaserSell 在本地创建新的密钥对。你需要在交易前为其充入 SOL。

无论哪种方式，你都需要设置加密密码短语（8 个以上字符）。此密码短语使用 XChaCha20Poly1305 在磁盘上加密你的私钥。LaserSell 从不以明文存储或传输你的密钥。

<Tip>
你可以稍后从[钱包 > 管理](/desktop/wallet)标签页添加更多钱包。你的账户等级决定了你可以同时使用的钱包数量。
</Tip>

## 步骤 3：初始策略

设置你的初始退出策略：

- **目标利润 %** — LaserSell 自动卖出的利润阈值（默认：25%）。
- **止损 %** — LaserSell 平仓的亏损阈值（默认：15%）。

这些是起始值。你可以在引导完成后从[退出策略](/desktop/exit-strategy)页面配置追踪止损、截止时间、毕业即卖、退出阶梯等更多选项。

## 步骤 4：RPC URL

可选设置自定义 Solana RPC 端点。如果你有来自 Helius、QuickNode、Triton 或其他提供商的专用 RPC，在此粘贴 URL 以获得更快更可靠的交易交付。

如果跳过此步骤，应用使用默认的公共端点。你可以稍后在[设置 > 连接](/desktop/settings)中更改。

## 步骤 5：崩溃报告

选择是否发送匿名崩溃报告。这有助于我们发现和修复 Bug。不会包含任何钱包数据、密钥或交易信息。你可以随时在设置中更改此选项。

## 解锁

引导完成后，后续启动时你会看到解锁界面。输入你的钱包密码短语来解密密钥并开始监控。

如果你有多个使用不同密码短语的钱包，可以分别解锁或跳过本次会话不需要的钱包。

在 macOS 上，你可以在[设置 > 安全](/desktop/security)中启用 **Touch ID** 以实现更快的解锁。

## 接下来会发生什么

解锁后，应用连接到 LaserSell 退出智能流并开始监控你的钱包。你持有的任何代币（或从此刻起购买的代币）将出现在[活跃仓位](/desktop/active-positions)仪表板中，带有实时盈亏追踪。
