LaserSell Desktop 是一款原生 macOS 应用，专为需要完全本地控制的交易者打造。它直接连接到 LaserSell 流，实时监控你所有钱包中的每个仓位，并在你的止盈、止损、追踪止损、截止时间或毕业即卖条件满足时自动执行卖出。你的密钥保存在你的设备上，每一笔交易都由你在本地签名。

无需管理浏览器标签页。无需批准钱包弹窗。你设置好策略，在你平常的平台上买入代币，让应用处理其余一切。

## 桌面应用的功能

1. **自动检测你的买入。** 你在任何平台买入代币的瞬间，应用就会识别并开始监控。
2. **在每次价格更新时评估你的退出策略。** 止盈、止损、追踪止损、截止时间和毕业即卖全部并行运行。最先触发的条件执行退出。
3. **自动卖出。** 你的退出条件一旦触发，LaserSell 会在 5 毫秒内完成交易构建，使用你的加密密钥在本地签名，并直接广播到 Solana 网络。一旦设置好策略，无需任何人工操作。这比第二快的方法快 12 倍以上（参见[基准测试](/benchmarks/tx-build-latency)）。
4. **保护你的密钥安全。** 每个钱包都使用只有你知道的密码短语在你的设备上加密。LaserSell 永远无法访问你的私钥。

## 主要功能

- **活跃仓位仪表板**，包含实时盈亏、市场数据和一键手动退出
- **退出策略配置**，支持单级或多级止盈阶梯、追踪止损和流动性保护
- **镜像交易**，从外部钱包跟单，每个钱包独立配置买入金额和表现追踪
- **内置钱包**，支持发送、接收、兑换和多钱包管理
- **数据分析**，包含每日盈亏图表、胜率、退出原因分布和个性化交易洞察
- **成就系统**，10 个类别，稀有度分级
- **Touch ID** 快速解锁 macOS 上的钱包
- **7 种语言**（英语、中文、日语、韩语、西班牙语、俄语、阿拉伯语）

## 支持的市场

应用监控并自动卖出所有主要 Solana 发射台和 DEX 上的代币：

- Pump.fun 和 PumpSwap
- Meteora DBC 和 DAMM V2
- Raydium Launchpad 和 CPMM

不支持的市场上的代币仍可通过内置的 Jupiter 集成手动兑换。

## 开始使用

1. [下载并安装](/desktop/installation)应用。
2. [登录并设置钱包](/desktop/onboarding)。
3. [配置你的退出策略](/desktop/exit-strategy)。
4. 在任何平台上买入代币。LaserSell 处理退出。

## 面向开发者

如果你想在 LaserSell 之上构建自己的应用而不是使用桌面应用，相同的退出智能引擎通过 [LaserSell SDK 和 API](/api/overview) 提供。桌面应用就是基于它构建的。
