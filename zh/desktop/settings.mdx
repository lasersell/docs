## 连接与账户

- **连接状态** — 显示应用是否已连接到 LaserSell 流（绿色 = 已连接，红色 = 已断开）。
- **账户等级** — 你当前的订阅：Personal（免费）、Professional 或 Advanced。
- **暂停监控** — 暂时停止所有流监控和自动卖出，而不断开连接。当你想手动持有仓位时很有用。

## 网络

### API 密钥

你的 LaserSell API 密钥。默认隐藏，可切换显示，带复制按钮。如需新密钥，使用登录流程生成。

### RPC URL

你的 Solana RPC 端点。如果你有来自 Helius、QuickNode 或 Triton 等提供商的专用 RPC，在此设置以获得更快更可靠的交易。默认为公共端点。

### 发送模式

LaserSell 如何将交易提交到网络：

- **RPC**（默认） — 通过你的 RPC 端点直接提交。无额外费用。
- **Helius Sender** — 通过 Helius 优化交易交付以获得更好的落地率。需要 Helius API 密钥和构建者小费（推荐：0.001 SOL）。
- **Astralane Iris** — MEV 保护的交易路由。需要 Astralane API 密钥和区域选择（法兰克福、阿姆斯特丹、纽约、旧金山、东京、新加坡等）。

## 显示

- **货币** — 选择投资组合价值和盈亏的显示货币。支持美洲、欧洲、亚太和中东的 20 种货币。
- **显示 USD1** — 在侧边栏显示 USD1 稳定币余额。
- **镜像 USD1 交易** — 启用 USD1 计价的跟单交易。
- **菜单栏图标** — 在 macOS 菜单栏显示 LaserSell 以便快速访问。
- **通知音效** — 交易执行时播放硬币音效。
- **启动时连接** — 应用启动时自动连接到流。如果你想将应用作为独立钱包使用而不进行流监控，可以禁用此选项。

## 交易

- **快速买入金额（SOL）** — 快速买入按钮的预设 SOL 金额（例如 0.1、0.5、1.0 SOL）。
- **快速买入金额（USD1）** — 预设 USD1 金额（启用 USD1 买入时显示）。
- **快速卖出百分比** — 部分卖出的预设百分比（例如 10%、25%、50%、100%）。
- **滑点容忍度** — 交易的最大允许价格滑点，以基点为单位。

## 语言与国际化支持

**支持的语言：**

- English
- Chinese (中文)
- Japanese (日本語)
- Korean (한국어)
- Spanish (Español)
- Russian (Русский)
- Arabic (العربية)

语言切换即时生效。每个页面、每个标签、每条通知都会立即更新，无需重启。

### 从右到左布局支持

LaserSell 完整支持阿拉伯语的从右到左（RTL）布局。当你切换到阿拉伯语时，整个界面会镜像翻转：导航栏移到右侧，文本变为从右到左对齐，每个组件都自然地重新排列。这不是简单的文字替换。整个布局，包括侧边栏、仓位列表、策略面板、钱包视图和分析图表，都会按照 RTL 阅读顺序重新构建。阿拉伯语用户将获得与所有其他语言同等精心打磨的原生体验。

### 多货币显示

你的投资组合价值、盈亏和交易金额可以用以下 20 种货币中的任意一种显示：

- **美洲** — USD、CAD
- **欧洲** — EUR、GBP、CHF
- **亚太** — JPY、CNY、KRW、INR、SGD、HKD、AUD、NZD
- **中东与非洲** — AED、SAR、KWD、BHD、QAR、OMR、EGP

所有数值会随汇率变化实时更新。选择你的本地货币，用对你最有意义的数字查看你的表现。

## 隐私

- **崩溃报告** — 切换通过 Sentry 发送匿名错误报告。不包含钱包数据或交易信息。

## 关于

- **版本** — 当前应用版本。
- **平台** — 你的操作系统。
- **获取帮助** — 支持资源链接。
- **复制错误报告** — 格式化最近的错误以便与支持团队分享。
- **注销** — 清除你的 API 密钥并返回登录界面。
