## 概览

选择时间周期（24 小时、7 天、30 天或全部时间）查看关键指标：

- **总盈亏** — 以你的显示货币计的净利润或亏损
- **总交易量** — 以报价单位计的总交易量
- **胜率** — 以盈利收盘的交易百分比
- **总交易笔数** — 已平仓位数量
- **平均盈亏** — 每笔交易的平均利润或亏损
- **平均持仓时间** — 仓位开仓的平均时间
- **最佳交易** — 你最高的单笔交易利润
- **最差交易** — 你最深的单笔交易亏损

### 每日盈亏图表

显示过去 90 天每日利润和亏损的柱状图。绿色柱代表盈利日，红色柱代表亏损日。将鼠标悬停在任意柱上查看确切数字。

### 退出原因分布

触发退出原因的可视化分布：

- **目标利润** — 策略达到止盈阈值
- **追踪止损** — 价格从峰值回落了追踪百分比
- **止损** — 价格达到止损阈值
- **毕业** — 代币从联合曲线迁移到 DEX
- **截止时间** — 仓位超时

这帮助你理解哪些退出条件做了最多工作，以及你的策略是否需要调整。

## 洞察（Tier 1+）

基于你最近交易模式的个性化建议。每个洞察都有严重程度级别（高、中、低）和具体建议。

洞察在服务器端从你的交易历史生成并定期更新。仅面向 Professional 及以上等级开放。

如果你在免费等级，你会看到升级提示。

## 成就

在 **10 个类别**中追踪你的进度，稀有度从普通到传说。

### 类别

| 类别 | 追踪内容 |
| :--- | :--- |
| 交易量 | 交易量里程碑 |
| 交易笔数 | 完成的交易数量 |
| 连续天数 | 连续盈利交易日 |
| 利润 | 累计利润目标 |
| 速度 | 快速执行时间 |
| 单笔交易 | 单笔交易盈亏记录 |
| 胜率 | 胜率百分比里程碑 |
| 镜像 | 跟单交易里程碑 |
| 秘密 | 隐藏成就（通过交易发现） |
| 精通 | 高级组合成就 |

### 稀有度等级

- **普通** — 入门级里程碑
- **不常见** — 中等成就
- **稀有** — 重要成就
- **史诗** — 卓越的交易里程碑
- **传说** — 系统中最稀有的成就

### 连续天数

你的当前和最长连续记录显示在成就标签页顶部。当你通过 LaserSell 协议路由的交易中有至少一笔盈利交易时，计为一个连续日。

### 成就如何运作

成就由服务器每小时检查。只有通过 LaserSell 协议路由的交易才计入成就进度。当你解锁新成就时，应用发送通知。

<Note>
解锁的成就未来将可兑换奖励。LaserSell 没有代币，也不会有空投。
</Note>
