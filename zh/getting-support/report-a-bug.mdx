## 安全漏洞

如果你发现任何 LaserSell 产品中的安全漏洞，请负责任地报告：

- **邮箱：** [security@lasersell.io](mailto:security@lasersell.io)
- **PGP：** 我们鼓励使用我们的 [PGP 密钥](https://www.lasersell.io/lasersell-pgp-key.asc)加密你的报告。完整披露指南请访问 [www.lasersell.io/.well-known/security.txt](https://www.lasersell.io/.well-known/security.txt)。

**不要**为安全漏洞创建公开的 GitHub Issue。

## 软件 Bug

对于 LaserSell 或 LaserSell SDK 中的 Bug，请使用以下渠道之一：

- **GitHub Issues：**
  - [lasersell/lasersell](https://github.com/lasersell/lasersell/issues)（客户端）
  - [lasersell/lasersell-sdk](https://github.com/lasersell/lasersell-sdk/issues)（SDK）
- **Discord 工单：** 在 [discord.gg/lasersell](https://discord.gg/lasersell) 中开启支持工单

报告时，请尽可能包含详细信息：复现步骤、错误消息、你的操作系统/平台和版本（`lasersell --version`）。

### 处理流程

我们致力于及时分类和修复已确认的 Bug。

- **通知：** 修复实施后，我们会通知你以便你获取最新更新。
- **更新：** 查看 [LaserSell 公告](https://t.me/lasersell) Telegram 频道或 Discord 的 `#announcements` 频道，确认修复已包含在你的版本中。
