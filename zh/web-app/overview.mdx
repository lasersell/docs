LaserSell Web App 将自动化交易带入你的浏览器。配置你的退出策略，为钱包充值，LaserSell 会代表你全天候执行交易，即使浏览器已关闭也不影响。

## 工作原理

<Steps>
  <Step title="设置钱包">
    登录后，[Privy](https://privy.io) 会自动为你创建一个安全钱包。你也可以导入现有钱包。你的私钥存储在可信执行环境（TEE）中，永远不会触及 LaserSell 服务器。
  </Step>
  <Step title="为钱包充值">
    从任何外部钱包（Phantom、Backpack 等）向你的钱包地址发送 SOL。你的钱包地址和余额可在钱包页面查看。
  </Step>
  <Step title="配置策略">
    设置你的退出策略：目标利润、止损、追踪止损和最大滑点。这些规则决定了 LaserSell 何时自动卖出你的仓位。
  </Step>
  <Step title="开始交易">
    点击「开始 Web 交易」，LaserSell 将开始监控你的钱包。当某个仓位达到你的策略阈值时，交易将使用你的钱包自动执行。
  </Step>
</Steps>

## Web App 与桌面应用对比

两个客户端连接到相同的 LaserSell 基础设施，但底层工作方式不同。

| 功能 | 桌面应用 | Web App |
|---------|------------|---------|
| **签名方式** | 本地密钥对 | Privy TEE 钱包（委托签名） |
| **离线执行** | 否（应用必须打开） | 是（服务器在你离开时执行） |
| **钱包** | 本地导入或生成 | Privy 内嵌或导入 |
| **跟单交易** | 是 | 是 |
| **仓位监控** | 是 | 是 |
| **买入代币** | 是 | 尚未支持 |
| **代币搜索** | 是 | 尚未支持 |

## 核心概念

### Privy 钱包

你的 Web 交易钱包由 [Privy](https://privy.io) 管理，Privy 是一家钱包基础设施公司，已被 [Stripe 收购](https://privy.io/blog/announcing-our-acquisition-by-stripe)。Privy 将你的私钥存储在可信执行环境（TEE）中，这是一个硬件隔离的安全区域，可以防止包括 LaserSell 在内的任何人直接访问你的密钥。

当需要执行交易时，LaserSell 请求 Privy 签署交易。Privy 在 TEE 内部完成签名并返回已签名的交易。然后 LaserSell 将其提交到 Solana 网络。

### 全天候监控

当 Web 交易处于活跃状态时，LaserSell 会代表你与监控基础设施保持持久连接。你的仓位被实时跟踪，当策略阈值被触发时，退出信号会立即执行。

你可以使用仪表板顶部的电源图标来开启或关闭 Web 交易。当切换到桌面应用时，请关闭 Web 交易以避免对同一钱包产生重复卖出操作。

### 会话状态

| 状态 | 含义 |
|-------|--------------|
| **活跃** | 流已连接，仓位受监控，交易自动执行 |
| **暂停** | 临时停止（错误或手动暂停） |
| **已停止** | Web 交易已关闭，不消耗资源 |
