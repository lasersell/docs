交易页面是 Web 交易的主控制中心。它有两个选项卡：**仓位** 用于监控你的活跃交易，**跟单交易** 用于镜像其他钱包。

## 开始使用

如果你尚未启动 Web 交易会话，交易页面会显示一个设置表单：

1. **选择要监控的钱包**（从你已注册的钱包中选择）
2. **配置退出策略**（目标利润、止损、追踪止损、最大滑点）
3. 点击 **开始 Web 交易**

启动后，LaserSell 会监控你的钱包，并在策略阈值被触发时自动执行交易。

## 仓位选项卡

仓位选项卡使用三栏布局：

- **左栏**：所有未平仓位列表及盈亏摘要
- **中栏**：所选仓位的详细视图（代币信息、入场价格、当前利润、卖出按钮）
- **右栏**：退出策略设置（可随时编辑）

### 退出策略设置

| 设置 | 说明 |
|---------|-------------|
| **目标利润 %** | 当利润达到此百分比时卖出 |
| **止损 %** | 当亏损达到此百分比时卖出 |
| **追踪止损 %** | 通过按此百分比追踪价格来锁定利润 |
| **最大滑点 %** | 自动卖出时可接受的最大滑点（默认 5%） |
| **毕业即卖** | 当代币从 pump.fun 毕业到 Raydium 时自动卖出 |

更改会立即应用到所有活跃仓位。

### 手动卖出

点击任何仓位上的 **立即卖出**，即可触发即时卖出，不受策略阈值限制。

## 跟单交易选项卡

跟单交易让你自动镜像其他钱包的交易操作。

### 添加观察钱包

1. 点击 **添加观察钱包**
2. 输入 Solana 钱包地址或 `.sol` 域名（通过 SNS 解析）
3. 设置昵称和自动买入金额（以 SOL 计）
4. 钱包将添加到你的观察列表中

当观察的钱包买入某个代币时，LaserSell 会自动使用你的钱包执行相同的买入操作。

### 镜像设置

| 设置 | 说明 |
|---------|-------------|
| **每个钱包最大仓位数** | 限制并发镜像仓位 |
| **冷却时间** | 镜像买入之间的最短间隔 |
| **跳过创建者代币** | 忽略被观察钱包创建的代币 |
| **最大活跃 SOL** | 限制镜像仓位的 SOL 总部署量 |
| **买入滑点** | 镜像买入的最大滑点 |
| **最低池流动性** | 跳过流动性较低的代币 |
| **最大入场偏移** | 如果价格相对被观察钱包的入场价偏移过大则跳过 |
| **最大连续亏损次数** | 连续亏损 N 次后自动禁用该观察钱包 |

## 会话控制

使用 **代理开关**（顶部栏中的电源图标）来启动或停止 Web 交易。状态指示点的含义：

- 🟢 **绿色**：活跃并正在监控
- 🔵 **蓝色**：等待新仓位
- 🟡 **黄色**：已暂停（错误或不活跃）
- ⚪ **灰色**：已停止

<Warning>
  如果你同时在桌面应用和 Web App 上监控同一个钱包，两者都会在阈值触发时尝试卖出。切换时请关闭其中一个。
</Warning>

## 等级限制

你的订阅等级决定了 Web 交易可用的容量。这些限制在桌面应用和 API 中同样适用。

| | 免费版 | 专业版 | 高级版 |
|--|------|-------------|----------|
| **钱包数** | 1 | 5 | 200 |
| **每个钱包的仓位数** | 2 | 100 | 100 |
| **观察钱包数** | 2 | 10 | 25 |
| **会话超时** | 45 分钟 | 无 | 无 |
| **全天候监控** | 否 | 是 | 是 |
| **退出阶梯** | 否 | 是 | 是 |
| **流动性保护** | 否 | 是 | 是 |
| **行为洞察** | 否 | 是 | 是 |

免费版用户可以完整使用 Web 交易的基础退出策略（目标利润、止损、追踪止损）。升级到专业版可获得高级策略、全天候监控和更大的容量。
