## 支持的发射台

| 平台 | 链接 |
|---|---|
| Pump.fun | [pump.fun](https://pump.fun) |
| Bonk.fun | [bonk.fun](https://bonk.fun) |
| Bags.fm | [bags.fm](https://bags.fm) |

| Trends.fun | [trends.fun](https://trends.fun) |
| Surge | [app.surge.xyz](https://app.surge.xyz/) |

使用以下支持协议的任何其他平台也可以工作。

## 支持的协议

| 协议 | 覆盖范围 |
|---|---|
| **Pump.fun 联合曲线** | 在 Pump.fun 联合曲线上买卖 |
| **PumpSwap** | 在 PumpSwap 市场上交易（毕业后） |
| **Raydium Launchpad** | Raydium 发射台上的代币（迁移前） |
| **Raydium CPMM** | 标准 Raydium 流动性池（迁移后） |
| **Meteora DBC** | Meteora 动态联合曲线（迁移前） |
| **Meteora DAMM V2** | Meteora 动态 AMM 池（迁移后） |

## 支持的资产

| 资产 | 详情 |
|---|---|
| **SOL** | 原生 SOL 和 Wrapped SOL（wSOL） |
| **SPL 代币** | 上述支持平台上的任何 SPL 代币 |
| **USD1** | USD1 稳定币交易对 |
