## المتطلبات

- **نقطة نهاية RPC خاصة.** نوصي بالتسجيل للحصول على RPC مجاني من [Helius](https://helius.dev) أو [Shyft](https://shyft.to) أو [Chainstack](https://chainstack.com). يمنحك RPC الخاص تحديثات أرصدة أسرع وتأكيدات معاملات أكثر موثوقية من RPC Solana العام.
- **مفتاح LaserSell API.** سجّل في [app.lasersell.io](https://app.lasersell.io) لإنشاء مفتاح API مجاني. المستوى المجاني كافٍ لمعظم المستخدمين.

## التثبيت

```shellscript
curl -fsSL https://dl.lasersell.io/install.sh | bash
```

تحقق من نجاحه:

```shellscript
lasersell --version
```

تحتاج تعليمات خاصة بالمنصة؟ راجع [نظرة عامة على التثبيت](/installation/overview).

## الإعداد

شغّل LaserSell. يبدأ معالج الإعداد تلقائياً عند أول تشغيل:

```shellscript
lasersell
```

سيطلب المعالج عنوان RPC ومفتاح API وإعدادات الاستراتيجية والمحفظة (إنشاء جديدة أو استيراد موجودة) وعبارة مرور لتشفير مخزن المفاتيح.

<Warning>
لا يوجد "نسيت كلمة المرور". إذا فقدت عبارة المرور، لا يمكن استعادة مخزن المفاتيح.
</Warning>

للاطلاع على شرح تفصيلي لكل مطالبة، راجع [الإعداد الأولي](/onboarding/initial-setup).

## التحقق

شغّل فحص صحة للتأكد من أن كل شيء متصل:

```shellscript
lasersell --smoke
```

يجب أن ترى `SMOKE OK`. ثم شغّل LaserSell:

```shellscript
lasersell
```

LaserSell يراقب محفظتك الآن.

## ما يحدث الآن

LaserSell هو **مراقب سلبي**. لا يضع صفقات نيابة عنك. بدلاً من ذلك، يراقب محفظتك بحثاً عن مشتريات رموز جديدة ويتعامل مع جانب البيع تلقائياً.

1. تشتري الرموز باستخدام أي منصة تتداول عليها بالفعل (Photon أو BullX أو Phantom أو Jupiter وغيرها)
2. يكتشف LaserSell الشراء تلقائياً خلال ثوانٍ
3. يسجل واجهة سطر الأوامر المركز كنشط مع تحديثات أرباح وخسائر حية
4. عندما ينطلق أي شرط خروج (جني الأرباح أو وقف الخسارة أو الوقف المتحرك أو الموعد النهائي أو التخرج)، يبني LaserSell ويوقع معاملة البيع تلقائياً
5. تستمر في التداول. يستمر LaserSell في البيع.

## أول خروج آلي لك

امشِ عبر الحلقة الكاملة لرؤية LaserSell أثناء العمل:

1. **LaserSell يعمل.** ترى مخرجات سجل تؤكد أن البث متصل ومحفظتك مسجلة.
2. **اشترِ مركزاً صغيراً** على أي منصة مدعومة. ابدأ صغيراً حتى تشاهد الآلية دون المخاطرة برأس مال حقيقي.
3. **شاهد السجلات.** خلال ثوانٍ، سترى أحداث `session_started` و`pnl_update` لمركزك الجديد.
4. **ينطلق الخروج.** عندما ينطلق شرط جني الأرباح أو وقف الخسارة أو الوقف المتحرك أو الموعد النهائي، سترى `sell_scheduled` يتبعه `sell_complete` مع التوقيع على السلسلة.

هذه هي الحلقة بأكملها. تتداول بشكل طبيعي ويتعامل LaserSell مع كل خروج.

## ضبط استراتيجيتك

عدّل `~/.lasersell/config.yml` لضبط استراتيجيتك ثم أعد تشغيل البرنامج. أو أعد تشغيل معالج الإعداد:

```shellscript
lasersell --setup
```

للمرجع الكامل للتكوين، راجع [مثال التكوين](/onboarding/example-configuration).
