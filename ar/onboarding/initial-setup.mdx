يعمل معالج الإعداد تلقائياً عند أول تشغيل. أعد تشغيله في أي وقت بـ `lasersell --setup`.

## ما تحتاجه

- **عنوان RPC** من [Helius](https://helius.dev) أو [Chainstack](https://chainstack.com) أو أي مزود Solana RPC
- **مفتاح LaserSell API** من [app.lasersell.io](https://app.lasersell.io) (مجاني)
- خطة محفظة: إنشاء محفظة جديدة أو تجهيز **عبارة الاسترداد** أو **المفتاح السري بصيغة base58** أو **ملف زوج مفاتيح Solana JSON**

## ما ينشئه المعالج

- `~/.lasersell/config.yml` (RPC ومفتاح API والاستراتيجية)
- `~/.lasersell/wallet.keystore.json` (محفظة مشفرة)

## شرح المعالج

### بيانات الاعتماد

```
RPC URL (private recommended): https://your-private-rpc.example.com
LaserSell API key: ********
```

إدخال مفتاح API مقنّع. يُنصح بشدة بـ RPC خاص للإنتاج.

### الاستراتيجية

يطالبك المعالج بتكوين كل شرط خروج. حدد القيم التي تتوافق مع أسلوب تداولك وتحملك للمخاطر.

- **جني الأرباح**: بيع عندما يصل الربح إلى هذه النسبة (مثل `6%`)
- **وقف الخسارة**: بيع عندما تصل الخسارة إلى هذه النسبة (مثل `10%`)
- **الوقف المتحرك**: بيع عندما ينخفض الربح بهذه النسبة من ذروته (مثل `5%`)
- **مهلة الموعد النهائي**: بيع إجباري بعد N ثانية (مثل `120`)
- **تحمل الانزلاق**: الحد الأقصى للانزلاق في عمليات البيع (مثل `20%`)
- **البيع عند التخرج**: بيع تلقائي عند انتقال الرمز إلى AMM (`y/N`)

يجب تفعيل شرط خروج واحد على الأقل (جني الأرباح أو وقف الخسارة أو الوقف المتحرك أو الموعد النهائي). يمكنك تغيير أي من هذه لاحقاً بتعديل `~/.lasersell/config.yml` وإعادة تشغيل البرنامج أو بإعادة تشغيل `lasersell --setup`.

### المحفظة

**تم اكتشاف مخزن مفاتيح موجود؟** يمكنك إعادة استخدامه أو إنشاء/استيراد محفظة مختلفة.

**إنشاء محفظة جديدة:** ينشئ LaserSell عبارة استرداد من 12 كلمة ويعرضها مرة واحدة. احفظها فوراً.

<Warning>
لن تُعرض عبارة الاسترداد مرة أخرى. إذا فقدتها، تفقد المحفظة.
</Warning>

**استيراد محفظة موجودة:** اختر واحداً من:

1. **مفتاح سري بصيغة base58** (الإدخال مقنّع)
2. **عبارة الاسترداد** (تستخدم مسار الاشتقاق `m/44'/501'/0'/0'`)
3. **مسار ملف زوج مفاتيح Solana JSON**

بعد الاستيراد، تحقق من أن المفتاح العام المعروض يطابق المحفظة التي قصدتها.

### عبارة المرور

حدد عبارة مرور لتشفير مخزن المفاتيح:

```
Keystore passphrase: ********
Confirm passphrase: ********
```

<Warning>
لا يوجد "نسيت كلمة المرور". إذا فقدت عبارة المرور، لا يمكن استعادة مخزن المفاتيح. مسار الاسترداد هو إعادة استيراد المحفظة من عبارة الاسترداد أو المفتاح الخاص.
</Warning>

### المراجعة والكتابة

يعرض LaserSell ملخصاً لإعداداتك ويطلب كتابة التكوين. أكّد للحفظ.

## بعد الإعداد

يستمر LaserSell في بدء التشغيل العادي. عند عمليات التشغيل المستقبلية، سيطلب عبارة مرور مخزن المفاتيح لفتح المحفظة.

للتشغيل بدون واجهة/الآلي، حدد متغير البيئة `LASERSELL_WALLET_PASSPHRASE`.

## استكشاف الأخطاء

**"config file not found; run --setup in an interactive terminal"**
يتطلب المعالج طرفية حقيقية. اتصل بالجهاز عبر SSH أو افتح طرفية محلية وشغّل `lasersell --setup`.

**عبارة الاسترداد المستوردة تعطي عنواناً خاطئاً**
يستخدم LaserSell مسار اشتقاق Solana القياسي (`m/44'/501'/0'/0'`). إذا كانت محفظتك تستخدم اشتقاقاً مختلفاً، استورد عبر المفتاح السري بصيغة base58 أو ملف زوج المفاتيح JSON بدلاً من ذلك.
