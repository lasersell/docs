مرحباً بك في السجل الرسمي لتطور منظومة **LaserSell**.

تعمل هذه الصفحة كأرشيف شامل لجميع إصدارات النسخ وتحديثات الميزات والتصحيحات الحرجة. بينما يتم تحديث هذا السجل مع كل إصدار رئيسي، نبث التحديثات في الوقت الفعلي عبر قنوات مجتمعنا.

### إصدارات GitHub

سجلات التغيير الكاملة لكل إصدار تُنشر على GitHub:

- **LaserSell (العميل):** [github.com/lasersell/lasersell/releases](https://github.com/lasersell/lasersell/releases)
- **حزمة تطوير LaserSell:** [github.com/lasersell/lasersell-sdk/releases](https://github.com/lasersell/lasersell-sdk/releases)

### الإعلانات الفورية

لتلقي إشعار فوري عند إصدار نسخة أو تصحيح جديد، انضم إلى قنوات الإعلانات:

- **Telegram:** [t.me/lasersell](https://t.me/lasersell)
- **Discord:** [discord.gg/lasersell](https://discord.gg/lasersell) (تحقق من قناتي `#announcements` و`#github`)
