راجع [النسخة الإنجليزية](/api/reference/market-context) للوثائق التقنية الكاملة مع مخطط `MarketContextMsg` وقيم `market_type` وكائنات السياق المتداخلة لكل بروتوكول (PumpFun وPumpSwap وMeteora DBC وMeteora DAMM V2 وRaydium Launchpad وRaydium CPMM) وتمرير سياق السوق في طلبات البناء ومرجع أنواع TypeScript.
