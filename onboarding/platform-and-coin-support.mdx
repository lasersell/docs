## Supported launchpads

| Platform | Link |
|---|---|
| Pump.fun | [pump.fun](https://pump.fun) |
| Bonk.fun | [bonk.fun](https://bonk.fun) |
| Bags.fm | [bags.fm](https://bags.fm) |
| Raydium LaunchLab | [raydium.io/launchpad](https://raydium.io/launchpad/) |
| Trends.fun | [trends.fun](https://trends.fun) |
| Surge | [app.surge.xyz](https://app.surge.xyz/) |

Any additional platform using a supported protocol below also works.

## Supported protocols

| Protocol | Coverage |
|---|---|
| **Pump.fun Bonding Curves** | Buy and sell on the Pump.fun bonding curve |
| **PumpSwap** | Trade on PumpSwap markets (post-graduation) |
| **Raydium Launchpad** | Tokens on the Raydium launchpad (pre-migration) |
| **Raydium CPMM** | Standard Raydium liquidity pools (post-migration) |
| **Meteora DBC** | Meteora Dynamic Bonding Curve (pre-migration) |
| **Meteora DAMM V2** | Meteora Dynamic AMM pools (post-migration) |

## Supported assets

| Asset | Details |
|---|---|
| **SOL** | Native SOL and Wrapped SOL (wSOL) |
| **SPL Tokens** | Any SPL token on the supported platforms above |
| **USD1** | USD1 stablecoin trading pairs |
