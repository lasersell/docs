After unlocking your wallet, LaserSell switches into the Terminal UI (TUI).

![Screenshot2026 01 28at6 41 59AM](/images/Screenshot2026-01-28at6.41.59AM.png)

## Layout

| Area | Purpose |
|------|---------|
| **Header** | Running/paused status, RPC latency, WebSocket status, wallet, SOL/USD1 balances |
| **Sessions** (left) | List of detected tokens with status |
| **Strategy** (right top) | Current strategy settings (TP, SL, TS, deadline, slippage, graduation) |
| **Selected** (right middle) | Details for the highlighted token (cost basis, PnL, market, proceeds) |
| **Output** (right bottom) | Command responses and log events |
| **Command** (bottom) | Type commands here |

Use **Tab** to switch focus between the Sessions pane and Command pane.

## Session statuses

| Status | Meaning |
|--------|---------|
| `DETECTED` | Mint detected, not yet started |
| `FILTERED` | Rejected by filters |
| `ACTIVE` | Position tracked, monitoring for exits |
| `SELL SCHED` | Exit triggered, sell queued |
| `SELLING` | Sell in progress |
| `CLOSED` | Sold or ended |
| `ERROR` | Failed |

## Keyboard controls

| Key | Action |
|-----|--------|
| **Tab / Shift+Tab** | Toggle focus between Sessions and Command |
| **Up / Down** | Move selection (Sessions pane) |
| **Enter** | Toggle expanded details (Sessions pane) |
| **Space** | Pause/resume new sessions (Sessions pane) |
| **m** | Toggle mouse capture (Sessions pane) |
| **Ctrl+C** | Quit |

## Commands

### Strategy commands

Update strategy settings live. Changes are saved to your config file immediately.

```
set tp 6%
set sl 10%
set ts 5%
set slippage 20%
set timeout 120
set graduation on
```

| Command | Aliases | Accepts |
|---------|---------|---------|
| `set tp <percent>` | | Percent string, e.g. `6%` |
| `set sl <percent>` | | Percent string, e.g. `10%` |
| `set ts <percent>` | `set trailing` | Percent string, e.g. `5%` |
| `set slippage <percent>` | `set slip` | Percent string, e.g. `20%` |
| `set timeout <seconds>` | `set to` | Integer seconds, `0` to disable |
| `set graduation <on\|off>` | | `on`/`true` or `off`/`false` |

### Trading commands

| Command | Aliases | Description |
|---------|---------|-------------|
| `sell` | `s` | Sell the selected position. Requires `y` confirmation. |

### Utility commands

| Command | Aliases | Description |
|---------|---------|-------------|
| `help` | `?` | Show available commands |
| `pause` | | Stop starting new sessions (existing ones continue) |
| `resume` | | Resume starting new sessions |
| `clear` | | Clear the output panel |
| `quit` | `exit`, `q` | Exit LaserSell |
