## Уязвимости безопасности

Если вы обнаружили уязвимость безопасности в любом продукте LaserSell, пожалуйста, сообщите об этом ответственно:

- **Email:** [security@lasersell.io](mailto:security@lasersell.io)
- **PGP:** Мы рекомендуем зашифровать ваш отчёт с помощью нашего [PGP ключа](https://www.lasersell.io/lasersell-pgp-key.asc). Полные рекомендации по раскрытию информации находятся на [www.lasersell.io/.well-known/security.txt](https://www.lasersell.io/.well-known/security.txt).

**Не** открывайте публичный issue на GitHub для уязвимостей безопасности.

## Программные ошибки

Для ошибок в LaserSell или LaserSell SDK используйте один из этих каналов:

- **GitHub Issues:**
  - [lasersell/lasersell](https://github.com/lasersell/lasersell/issues) (клиент)
  - [lasersell/lasersell-sdk](https://github.com/lasersell/lasersell-sdk/issues) (SDK)
- **Тикет в Discord:** Откройте тикет поддержки в [discord.gg/lasersell](https://discord.gg/lasersell)

При сообщении об ошибке включайте как можно больше деталей: шаги для воспроизведения, сообщения об ошибках, вашу ОС/платформу и версию (`lasersell --version`).

### Процесс решения

Мы стремимся оперативно классифицировать и исправлять подтверждённые ошибки.

- **Уведомление:** После реализации исправления мы уведомим вас, чтобы вы могли загрузить последнее обновление.
- **Обновления:** Проверяйте Telegram канал [LaserSell Announcements](https://t.me/lasersell) или канал `#announcements` в Discord, чтобы убедиться, что исправление включено в вашу сборку.
