LaserSell отслеживает ваши торговые вехи и присваивает достижения по мере их достижения. Существует более 60 достижений в 10 категориях, от обычных вех, которых может достичь каждый, до легендарных достижений, требующих исключительной результативности.

<Note>
Разблокированные достижения можно будет обменять на награды, не имеющие аналогов среди того, что ранее предлагалось трейдерам. Токена LaserSell не существует и не будет аирдропа.
</Note>

## Как работают достижения

Достижения проверяются автоматически с часовой периодичностью. Только сделки, проведённые через протокол LaserSell, засчитываются в прогресс достижений. Когда вы открываете новое достижение, приложение отправляет уведомление.

Скрытые достижения не раскрывают своих условий, пока вы их не разблокируете.

## Уровни редкости

| Редкость | Описание |
| :--- | :--- |
| **Обычное** | Начальные вехи. Вы разблокируете их естественным образом, когда начнёте торговать. |
| **Необычное** | Умеренные достижения, требующие устойчивой торговой активности. |
| **Редкое** | Значительные достижения, требующие реальных усилий или мастерства. |
| **Эпическое** | Исключительные вехи, достигнутые небольшим процентом трейдеров. |
| **Легендарное** | Самые редкие достижения в системе. Они требуют экстраординарного объёма, последовательности или единичного момента исключительной результативности. |

## Категории

### Объём

Вехи по совокупному торговому объёму. Чем больше вы торгуете через LaserSell, тем выше поднимаетесь.

### Сделки

Вехи по общему количеству завершённых сделок, от вашей первой сделки до тысяч.

### Серия

Вехи по последовательным дням торговли. Торговый день засчитывается, когда вы совершаете хотя бы одну сделку через протокол LaserSell. Пропуск дня сбрасывает вашу текущую серию.

Ваша текущая серия и самая длинная серия отображаются в верхней части вкладки «Достижения» в приложении.

### Прибыль

Вехи по совокупной реализованной прибыли в SOL.

### Скорость

Достижения за прибыльные сделки, совершённые в короткие сроки. Они награждают быстрое исполнение и скорые выходы.

### Одиночная сделка

Достижения на основе результативности одной позиции, от хороших доходов до экстраординарных мультипликаторов.

### Процент побед

Вехи по проценту побед с минимальными требованиями по количеству сделок для предотвращения манипуляций. Поддержание высокого процента побед на протяжении сотен сделок — одна из самых сложных задач.

### Зеркало

Достижения за [зеркальную торговлю](/desktop/mirror-trading). Отслеживают вашу активность и результативность копирования сделок по отслеживаемым кошелькам.

### Секретные

Скрытые достижения с условиями, которые не раскрываются до разблокировки. Вы откроете их естественным образом в процессе торговли.

### Мастерство

Мета-достижения, присуждаемые за выполнение всех достижений в категории. Для каждой из остальных категорий есть одно достижение «Мастерство». Выполнение всех секретных достижений приносит единственное легендарное «Мастерство».

## Серии

Ваша торговая серия отслеживает последовательные дни с хотя бы одной завершённой сделкой через протокол LaserSell.

- **Текущая серия** — сколько дней подряд вы торгуете, включая сегодня.
- **Самая длинная серия** — ваш рекорд за всё время.
- **Дата последней сделки** — когда была записана ваша последняя сделка.

Пропуск одного дня сбрасывает вашу текущую серию до нуля. Самая длинная серия сохраняется навсегда.

## Доступ через API

Достижения доступны программно:

```
GET /v1/achievements
```

Возвращает все достижения с их статусом разблокировки и временными метками. Заблокированные достижения в категории «Секретные» возвращают скрытые описания.
