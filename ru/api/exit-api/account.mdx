## Эндпоинт

```
GET https://api.lasersell.io/v1/account
```

## Заголовки

| Заголовок | Обязательный | Описание |
|-------------|----------|------------------------|
| `x-api-key` | Да | Ваш API ключ LaserSell |

## Ответ: `AccountResponse`

| Поле | Тип | Описание |
|-------------|----------|--------------------------------------------------|
| `tier` | `number` | Числовой идентификатор тарифа (`0`, `1` или `2`). |
| `tier_name` | `string` | Читаемое название плана (`free`, `tier_1`, `tier_2`). |
| `limits` | `object` | Лимиты использования для вашего плана (см. ниже). |

### Объект `limits`

| Поле | Тип | Описание |
|-----------------------------|----------|-------------------------------------------------------|
| `max_wallets` | `number` | Максимальное количество кошельков для одновременного отслеживания. |
| `max_positions_per_wallet` | `number` | Максимальное количество открытых позиций на кошелёк. |
| `max_positions_per_session` | `number` | Максимальное общее количество открытых позиций по всем кошелькам. |

## Пример curl

```bash
curl https://api.lasersell.io/v1/account \
  -H "x-api-key: YOUR_API_KEY"
```

## Пример ответа

```json
{
  "tier": 1,
  "tier_name": "tier_1",
  "limits": {
    "max_wallets": 5,
    "max_positions_per_wallet": 100,
    "max_positions_per_session": 500
  }
}
```
