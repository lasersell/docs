Навыки кодирования дают вашему ИИ ассистенту глубокие встроенные знания о LaserSell SDK. Вместо поиска документации по запросу ассистент загружает паттерны SDK, сигнатуры методов, рабочие примеры, конфигурации стратегий и руководства по устранению неполадок непосредственно в своё контекстное окно. Это означает, что он может писать корректный код LaserSell с самого начала, без необходимости что-либо искать.

## Навыки vs MCP

LaserSell предлагает два пути интеграции с ИИ. Они служат разным целям и лучше всего работают вместе.

| | MCP сервер | Навыки кодирования |
|---|---|---|
| **Как работает** | Ищет документацию LaserSell по запросу, когда ассистенту нужно что-то найти | Загружает глубокие знания SDK непосредственно в контекст ассистента до начала написания кода |
| **Лучше всего для** | Поиск схем API, нахождение конкретных страниц документации, ответы на вопросы о LaserSell | Написание интеграций с SDK, построение транзакций, подключение к потоку, обработка ошибок |
| **Когда знания доступны** | По запросу (ассистент должен запросить) | Немедленно (ассистент уже имеет их в контексте) |
| **Настройка** | Добавить конфигурацию MCP сервера в редактор | Установить пакет навыка |

MCP сервер отлично подходит для обнаружения и справочных поисков. Навыки кодирования отлично подходят для написания кода, интегрирующегося с SDK. Используйте оба для лучшего опыта.

## Что включено

Навык кодирования LaserSell поставляет структурированное дерево знаний, которое ассистент загружает при работе с кодом LaserSell:

- **SKILL.md** — корневой файл навыка с обзором SDK, ключевыми концепциями и указателями на детальные справочники
- **Рабочие примеры на 4 языках** — примеры на TypeScript, Python, Rust и Go, охватывающие транзакции покупки/продажи, подключения к потоку, конфигурацию стратегии и отправку транзакций
- **Руководство по стратегии** — детальный справочник по конфигурации тейк профита, стоп лосса, трейлинг стопа, продажи при выпуске на DEX и дедлайна
- **Руководство по устранению неполадок** — распространённые ошибки, их причины и способы исправления
- **Справочник API и типов** — схемы запросов/ответов, значения перечислений и описания полей для каждого эндпоинта
- **Справочник протокола потока** — полный формат сообщений Exit Intelligence Stream, типы событий и жизненный цикл соединения
- **Шаблоны** — стартовые шаблоны кода для распространённых паттернов интеграции

## Установка

### Вариант 1: npx (рекомендуемый)

```bash
npx skills add lasersell/skills
```

Это загружает и устанавливает навык кодирования LaserSell в ваш проект. Ассистент подхватит его автоматически при следующей сессии.

### Вариант 2: Claude Code CLI

```bash
claude install-skill https://github.com/lasersell/skills/tree/main/skills/lasersell
```

### Вариант 3: Вручную

Скопируйте директорию `skills/lasersell/` из [репозитория навыков LaserSell](https://github.com/lasersell/skills) в папку `.claude/skills/` вашего проекта:

```bash
mkdir -p .claude/skills
cp -r path/to/skills/lasersell .claude/skills/
```

Ассистент обнаружит файлы навыка и загрузит их в контекст при работе с кодом LaserSell.

## Что охватывает навык

После установки ваш ИИ ассистент имеет полный контекст по следующим темам:

**Построение транзакций покупки и продажи.** Ассистент знает, как создавать объекты `BuildBuyTxRequest` и `BuildSellTxRequest`, вызывать LaserSell API и обрабатывать ответы на всех четырёх языках SDK. Он понимает типы вывода (SOL vs USD1), конфигурацию проскальзывания и частичные продажи.

**Exit Intelligence Stream.** Ассистент может писать код подключения к потоку, настраивать параметры стратегии, обрабатывать все типы серверных событий (`pnl_update`, `exit_signal_with_tx`, `position_opened`, `position_closed`, `liquidity_snapshot`) и обрабатывать сигналы выхода от начала до конца.

**Частичные и многоэтапные продажи.** Ассистент понимает, как запрашивать полосы проскальзывания, определять максимальный объём продажи при заданном пороге проскальзывания и строить транзакции частичной продажи. Он также может создавать многоэтапные последовательности продажи для крупных позиций.

**Снимки ликвидности.** Ассистент знает, как читать полосы проскальзывания, интерпретировать тренды ликвидности (`growing`, `stable`, `draining`) и использовать данные снимков для оптимального определения размера продаж.

**Подпись и отправка транзакций.** Ассистент может писать код для локальной подписи транзакций и понимает все цели отправки: default RPC, custom RPC и Helius Sender. Он знает, когда использовать каждую и как их настраивать.

**Обработка ошибок.** Ассистент имеет контекст по распространённым кодам ошибок, паттернам повторных попыток, сбоям проскальзывания и восстановлению соединения как для REST API, так и для потока.

## Совместное использование навыков и MCP

Для лучшего опыта ИИ кодирования используйте навык кодирования и MCP сервер вместе:

1. **Установите навык кодирования**, чтобы ваш ассистент имел глубокие знания SDK, загруженные в контекст при каждом написании кода LaserSell.
2. **Подключите MCP сервер**, чтобы ваш ассистент мог искать полную документацию, когда ему нужно найти что-то за пределами охвата навыка, например, детали партнёрских комиссий, тарифные лимиты или записи журнала изменений.

С обоими активными ассистент может писать корректный код SDK немедленно (из навыка) и искать дополнительную информацию по запросу (из MCP).

См. [MCP сервер](/ai-agents/mcp-server) для инструкций по настройке MCP.
