Когда вы открываете LaserSell Desktop впервые, мастер настройки проведёт вас через пять шагов. Весь процесс занимает около двух минут.

## Шаг 1: Вход

Войдите в свой аккаунт LaserSell для получения API ключа. У вас есть два варианта:

- **Вход через браузер:** Нажмите «Sign In» и пройдите аутентификацию в браузере. Приложение автоматически получит ваш API ключ через deep link.
- **Ручная вставка:** Если у вас уже есть API ключ, вставьте его напрямую.

Если у вас ещё нет аккаунта, нажмите «Sign Up», чтобы создать его. Аккаунты бесплатные и создаются мгновенно.

## Шаг 2: Настройка кошелька

Вам нужен хотя бы один кошелёк Solana для мониторинга LaserSell. Выберите один вариант:

- **Импорт существующего кошелька.** Вставьте ваш приватный ключ Base58. Это удобно, если у вас уже есть пополненный кошелёк, которым вы торгуете.
- **Создание нового кошелька.** LaserSell создаёт свежую пару ключей локально. Вам нужно будет пополнить его SOL перед торговлей.

В любом случае вы установите парольную фразу шифрования (8+ символов). Эта парольная фраза шифрует ваш приватный ключ на диске с помощью XChaCha20Poly1305. LaserSell никогда не хранит и не передаёт ваши ключи в открытом виде.

<Tip>
Вы можете добавить больше кошельков позже из вкладки [Кошелёк > Управление](/desktop/wallet). Ваш тарифный план определяет, сколько кошельков вы можете использовать одновременно.
</Tip>

## Шаг 3: Начальная стратегия

Установите вашу начальную стратегию выхода:

- **Тейк профит %** — порог прибыли, при котором LaserSell автоматически продаёт (по умолчанию: 25%).
- **Стоп лосс %** — порог убытка, при котором LaserSell закрывает позицию (по умолчанию: 15%).

Это начальные значения. Вы можете настроить трейлинг стопы, дедлайны, продажу при выпуске на DEX, лестницу выхода и многое другое на странице [Стратегия выхода](/desktop/exit-strategy) после настройки.

## Шаг 4: URL RPC

Необязательно укажите свой эндпоинт Solana RPC. Если у вас есть выделенный RPC от Helius, QuickNode, Triton или другого провайдера, вставьте URL сюда для более быстрой и надёжной доставки транзакций.

Если вы пропустите этот шаг, приложение использует публичный эндпоинт по умолчанию. Вы можете изменить его позже в [Настройки > Подключение](/desktop/settings).

## Шаг 5: Отчёты о сбоях

Выберите, отправлять ли анонимные отчёты о сбоях. Это помогает нам находить и исправлять ошибки. Данные кошелька, ключи или торговая информация никогда не включаются. Вы можете изменить это в любое время в Настройках.

## Разблокировка

После настройки при последующих запусках вы увидите экран разблокировки. Введите парольную фразу кошелька для расшифровки ключей и начала мониторинга.

Если у вас несколько кошельков с разными парольными фразами, вы можете разблокировать их по отдельности или пропустить кошельки, которые вам не нужны для этой сессии.

На macOS вы можете включить **Touch ID** в [Настройки > Безопасность](/desktop/security) для более быстрой разблокировки.

## Что происходит далее

После разблокировки приложение подключается к потоку Exit Intelligence от LaserSell и начинает мониторинг ваших кошельков. Все токены, которые вы держите (или купите с этого момента), появятся на панели [Активные позиции](/desktop/active-positions) с отслеживанием PnL в реальном времени.
