## Подключение и аккаунт

- **Статус подключения** — показывает, подключено ли приложение к потоку LaserSell (зелёный = подключено, красный = отключено).
- **Тариф аккаунта** — ваш текущий тариф: Personal (бесплатный), Professional или Advanced.
- **Приостановить мониторинг** — временно остановить весь мониторинг потока и автопродажи без отключения. Полезно, когда вы хотите управлять позициями вручную.

## Сеть

### API ключ

Ваш API ключ LaserSell. Скрыт по умолчанию с переключателем для отображения и кнопкой копирования. Если вам нужен новый ключ, используйте процесс входа для его генерации.

### URL RPC

Ваш эндпоинт Solana RPC. Если у вас есть выделенный RPC от провайдера вроде Helius, QuickNode или Triton, установите его здесь для более быстрых и надёжных транзакций. По умолчанию используется публичный эндпоинт.

### Режим отправки

Как LaserSell отправляет транзакции в сеть:

- **RPC** (по умолчанию) — прямая отправка через ваш RPC эндпоинт. Без дополнительных комиссий.
- **Helius Sender** — оптимизированная доставка транзакций через Helius для лучших показателей попадания. Требуется API ключ Helius и сборщик (рекомендуется: 0.001 SOL).
- **Astralane Iris** — MEV-защищённая маршрутизация транзакций. Требуется API ключ Astralane и выбор региона (Франкфурт, Амстердам, Нью-Йорк, Сан-Франциско, Токио, Сингапур и другие).

## Отображение

- **Валюта** — выберите валюту отображения для стоимости портфеля и PnL. Поддерживается 20 валют из Америк, Европы, Азиатско-Тихоокеанского региона и Ближнего Востока.
- **Показать USD1** — отображать баланс стейблкоина USD1 на боковой панели.
- **Зеркалирование USD1** — включить деноминированное в USD1 копирование сделок.
- **Иконка в панели меню** — показать LaserSell в панели меню macOS для быстрого доступа.
- **Звук уведомления** — воспроизводить звук монеты при исполнении сделок.
- **Подключение при запуске** — автоматически подключаться к потоку при запуске приложения. Отключите это, если хотите использовать приложение как автономный кошелёк без мониторинга потока.

## Торговля

- **Суммы быстрой покупки (SOL)** — предустановленные суммы SOL для кнопок быстрой покупки (например, 0.1, 0.5, 1.0 SOL).
- **Суммы быстрой покупки (USD1)** — предустановленные суммы USD1 (показываются при включении покупки в USD1).
- **Процент быстрой продажи** — предустановленные проценты продажи для частичных продаж (например, 10%, 25%, 50%, 100%).
- **Допуск проскальзывания** — максимально допустимое проскальзывание цены при сделках, в базисных пунктах.

## Язык и международная поддержка

LaserSell создан для трейдеров по всему миру. Весь интерфейс приложения полностью локализован на 7 языков, и макет автоматически адаптируется к каждому из них.

**Поддерживаемые языки:**

- English
- Chinese (中文)
- Japanese (日本語)
- Korean (한국어)
- Spanish (Español)
- Russian (Русский)
- Arabic (العربية)

Переключение языка происходит мгновенно. Каждый экран, каждая надпись, каждое уведомление обновляются сразу, без необходимости перезапуска.

### Поддержка письма справа налево

LaserSell полностью поддерживает макеты справа налево (RTL) для арабского языка. При переключении на арабский весь интерфейс зеркально отражается: навигация перемещается вправо, текст выравнивается справа налево, и каждый компонент перестраивается естественным образом. Это не простая замена текста. Весь макет, включая боковую панель, списки позиций, панели стратегий, представления кошелька и графики аналитики, перестраивается для порядка чтения RTL. Арабоязычные трейдеры получают тот же качественный, нативный опыт, что и на любом другом языке.

### Мультивалютное отображение

Стоимость портфеля, PnL и суммы сделок могут отображаться в любой из 20 валют:

- **Америка** — USD, CAD
- **Европа** — EUR, GBP, CHF
- **Азиатско-Тихоокеанский регион** — JPY, CNY, KRW, INR, SGD, HKD, AUD, NZD
- **Ближний Восток и Африка** — AED, SAR, KWD, BHD, QAR, OMR, EGP

Все значения обновляются в реальном времени по мере изменения курсов валют. Выберите свою местную валюту и отслеживайте результаты в привычных цифрах.

## Конфиденциальность

- **Отчёты о сбоях** — переключатель анонимных отчётов об ошибках через Sentry. Данные кошелька или торговая информация не включаются.

## О приложении

- **Версия** — текущая версия приложения.
- **Платформа** — ваша операционная система.
- **Получить помощь** — ссылка на ресурсы поддержки.
- **Скопировать отчёт об ошибке** — форматирует последние ошибки для отправки в поддержку.
- **Выйти** — очищает ваш API ключ и возвращает на экран входа.
