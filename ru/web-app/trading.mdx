Страница торговли является основным центром управления веб торговлей. Она содержит две вкладки: **Positions** для мониторинга активных сделок и **Copy Trading** для зеркального копирования других кошельков.

## Начало работы

Если вы ещё не запускали сессию веб торговли, на странице торговли отображается форма настройки:

1. **Выберите кошельки** для мониторинга (из ваших зарегистрированных кошельков)
2. **Настройте стратегию выхода** (целевая прибыль, стоп лосс, трейлинг стоп, максимальное проскальзывание)
3. Нажмите **Start Web Trading**

После запуска LaserSell отслеживает ваши кошельки и автоматически исполняет сделки при достижении пороговых значений вашей стратегии.

## Вкладка позиций

Вкладка позиций использует трёхколоночный макет:

- **Слева**: Список всех открытых позиций с итогами PnL
- **По центру**: Подробный вид выбранной позиции (информация о токене, цена входа, текущая прибыль, кнопка продажи)
- **Справа**: Настройки стратегии выхода (редактируемые в любое время)

### Настройки стратегии выхода

| Настройка | Описание |
|---------|-------------|
| **Target Profit %** | Продавать, когда прибыль достигает этого процента |
| **Stop Loss %** | Продавать, когда убыток достигает этого процента |
| **Trailing Stop %** | Фиксировать прибыль, следуя за ценой на этот процент |
| **Max Slippage %** | Максимально допустимое проскальзывание при автоматических продажах (по умолчанию 5%) |
| **Sell on Graduation** | Автоматическая продажа при переходе токена с pump.fun на Raydium |

Изменения применяются немедленно ко всем активным позициям.

### Ручная продажа

Нажмите **Sell Now** на любой позиции, чтобы инициировать немедленную продажу, независимо от пороговых значений стратегии.

## Вкладка Copy Trading

Копирование сделок позволяет автоматически зеркально повторять сделки других кошельков.

### Добавление наблюдаемого кошелька

1. Нажмите **Add Watch Wallet**
2. Введите адрес кошелька Solana или доменное имя `.sol` (разрешается через SNS)
3. Задайте никнейм и сумму автоматической покупки (в SOL)
4. Кошелёк добавляется в ваш список наблюдения

Когда наблюдаемый кошелёк покупает токен, LaserSell автоматически выполняет ту же покупку вашим кошельком.

### Настройки зеркальной торговли

| Настройка | Описание |
|---------|-------------|
| **Max positions per wallet** | Ограничение параллельных зеркальных позиций |
| **Cooldown** | Минимальное время между зеркальными покупками |
| **Skip creator tokens** | Игнорировать токены, созданные наблюдаемым кошельком |
| **Max active SOL** | Ограничить общий объём SOL, распределённого по зеркальным позициям |
| **Buy slippage** | Максимальное проскальзывание для зеркальных покупок |
| **Min pool liquidity** | Пропускать токены с низкой ликвидностью |
| **Max entry drift** | Пропускать, если цена слишком отклонилась от цены входа наблюдаемого кошелька |
| **Max consecutive losses** | Автоматически отключить наблюдаемый кошелёк после N последовательных убытков |

## Управление сессией

Используйте **переключатель агента** (значок питания в заголовке), чтобы запустить или остановить веб торговлю. Индикатор состояния показывает:

- 🟢 **Зелёный**: Активна, идёт мониторинг
- 🔵 **Синий**: Ожидание новых позиций
- 🟡 **Жёлтый**: Приостановлена (ошибка или бездействие)
- ⚪ **Серый**: Остановлена

<Warning>
  Если вы отслеживаете один и тот же кошелёк и с десктопного, и с веб приложения, оба попытаются продать при достижении порога. Отключите одно из них при переключении на другое.
</Warning>

## Лимиты по тарифам

Ваш тарифный план определяет доступные возможности для веб торговли. Те же лимиты действуют в десктопном приложении и API.

| | Free | Professional | Advanced |
|--|------|-------------|----------|
| **Кошельки** | 1 | 5 | 200 |
| **Позиций на кошелёк** | 2 | 100 | 100 |
| **Наблюдаемые кошельки** | 2 | 10 | 25 |
| **Таймаут сессии** | 45 мин | Нет | Нет |
| **Непрерывный мониторинг** | Нет | Да | Да |
| **Exit Ladder** | Нет | Да | Да |
| **Liquidity Guard** | Нет | Да | Да |
| **Behavioral Insights** | Нет | Да | Да |

Пользователи бесплатного тарифа имеют полный доступ к веб торговле с базовыми стратегиями выхода (целевая прибыль, стоп лосс, трейлинг стоп). Перейдите на Professional для доступа к продвинутым стратегиям, непрерывному мониторингу и увеличенным лимитам.
