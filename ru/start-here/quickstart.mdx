## Предварительные требования

- **Частный RPC эндпоинт.** Мы рекомендуем зарегистрировать бесплатный RPC от [Helius](https://helius.dev), [Shyft](https://shyft.to) или [Chainstack](https://chainstack.com). Частный RPC обеспечивает более быстрые обновления баланса и более надёжные подтверждения транзакций, чем публичный RPC Solana.
- **API ключ LaserSell.** Зарегистрируйтесь на [app.lasersell.io](https://app.lasersell.io), чтобы создать бесплатный API ключ. Бесплатного тарифа достаточно для большинства пользователей.

## Установка

```shellscript
curl -fsSL https://dl.lasersell.io/install.sh | bash
```

Проверьте, что всё работает:

```shellscript
lasersell --version
```

Нужны инструкции для конкретной платформы? См. [Обзор установки](/installation/overview).

## Настройка

Запустите LaserSell. Мастер настройки запускается автоматически при первом запуске:

```shellscript
lasersell
```

Мастер запросит URL RPC, API ключ, настройки стратегии, кошелёк (создать новый или импортировать существующий) и парольную фразу для шифрования хранилища.

<Warning>
Функции «забыли пароль» не существует. Если вы потеряете парольную фразу, хранилище невозможно восстановить.
</Warning>

Подробное описание каждого шага см. в [Начальная настройка](/onboarding/initial-setup).

## Проверка

Запустите проверку работоспособности, чтобы убедиться, что всё подключено:

```shellscript
lasersell --smoke
```

Вы должны увидеть `SMOKE OK`. Затем запустите LaserSell:

```shellscript
lasersell
```

LaserSell теперь мониторит ваш кошелёк.

## Что происходит дальше

LaserSell — это **пассивный монитор**. Он не совершает сделки за вас. Вместо этого он наблюдает за вашим кошельком для обнаружения новых покупок токенов и автоматически обрабатывает сторону продажи.

1. Вы покупаете токены на любой платформе, которой уже пользуетесь
2. LaserSell автоматически обнаруживает покупку в течение секунд
3. CLI логирует позицию как активную с обновлениями PnL в реальном времени
4. Когда любое условие выхода срабатывает (тейк профит, стоп лосс, трейлинг стоп, дедлайн или выпуск на DEX), LaserSell автоматически строит и подписывает транзакцию продажи
5. Вы продолжаете торговать. LaserSell продолжает продавать.

## Ваш первый автоматический выход

Пройдите полный цикл, чтобы увидеть LaserSell в действии:

1. **LaserSell работает.** Вы видите логи, подтверждающие, что поток подключён и ваш кошелёк зарегистрирован.
2. **Купите маленькую позицию** на любой поддерживаемой платформе. Начните с малого, чтобы можно было наблюдать механику без риска реального капитала.
3. **Смотрите логи.** В течение секунд вы увидите события `session_started` и `pnl_update` для вашей новой позиции.
4. **Выход срабатывает.** Когда сработает тейк профит, стоп лосс, трейлинг стоп или дедлайн, вы увидите `sell_scheduled`, за которым следует `sell_complete` с подписью в блокчейне.

Это весь цикл. Вы торгуете как обычно, а LaserSell обрабатывает каждый выход.

## Настройка стратегии

Отредактируйте `~/.lasersell/config.yml` для настройки вашей стратегии, затем перезапустите демон. Или запустите мастер настройки заново:

```shellscript
lasersell --setup
```

Полный справочник по конфигурации см. в [Пример конфигурации](/onboarding/example-configuration).
