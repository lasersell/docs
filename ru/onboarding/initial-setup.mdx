Мастер настройки запускается автоматически при первом запуске. Запустите его снова в любое время командой `lasersell --setup`.

## Что вам понадобится

- **URL RPC** от [Helius](https://helius.dev), [Chainstack](https://chainstack.com) или любого провайдера Solana RPC
- **API ключ LaserSell** с [app.lasersell.io](https://app.lasersell.io) (бесплатно)
- План кошелька: создать новый кошелёк или подготовить **сид-фразу**, **base58 приватный ключ** или **JSON-файл keypair Solana** для импорта

## Что создаёт мастер

- `~/.lasersell/config.yml` (RPC, API ключ, стратегия)
- `~/.lasersell/wallet.keystore.json` (зашифрованный кошелёк)

## Пошаговое описание мастера

### Учётные данные

```
RPC URL (private recommended): https://your-private-rpc.example.com
LaserSell API key: ********
```

Ввод API ключа замаскирован. Частный RPC настоятельно рекомендуется для продакшена.

### Стратегия

Мастер предлагает вам настроить каждое условие выхода. Установите значения, соответствующие вашему торговому стилю и допустимому риску.

- **Тейк профит**: продажа при достижении этого % прибыли (например, `6%`)
- **Стоп лосс**: продажа при достижении этого % убытка (например, `10%`)
- **Трейлинг стоп**: продажа при падении прибыли на этот % от пика (например, `5%`)
- **Дедлайн**: принудительная продажа через N секунд (например, `120`)
- **Допуск проскальзывания**: максимальное проскальзывание для продаж (например, `20%`)
- **Продажа при выпуске на DEX**: автопродажа при миграции токена на AMM (`y/N`)

Должно быть включено хотя бы одно условие выхода (тейк профит, стоп лосс, трейлинг стоп или дедлайн). Вы можете изменить любое из них позже, отредактировав `~/.lasersell/config.yml` и перезапустив демон, или повторно запустив `lasersell --setup`.

### Кошелёк

**Обнаружено существующее хранилище?** Вы можете использовать его повторно или создать/импортировать другой кошелёк.

**Создание нового кошелька:** LaserSell генерирует сид-фразу из 12 слов и отображает её один раз. Сохраните её немедленно.

<Warning>
Сид-фраза не будет показана повторно. Если вы её потеряете, вы потеряете кошелёк.
</Warning>

**Импорт существующего кошелька:** Выберите один из вариантов:

1. **Base58 приватный ключ** (ввод замаскирован)
2. **Сид-фраза** (используется путь деривации `m/44'/501'/0'/0'`)
3. **JSON-файл keypair Solana** (путь к файлу)

После импорта проверьте, что отображаемый pubkey соответствует тому кошельку, который вы намеревались использовать.

### Парольная фраза

Установите парольную фразу для шифрования хранилища:

```
Keystore passphrase: ********
Confirm passphrase: ********
```

<Warning>
Функции «забыли пароль» не существует. Если вы потеряете парольную фразу, хранилище невозможно восстановить. Ваш путь восстановления — повторный импорт кошелька из сид-фразы или приватного ключа.
</Warning>

### Просмотр и сохранение

LaserSell показывает сводку ваших настроек и предлагает записать конфигурацию. Подтвердите для сохранения.

## После настройки

LaserSell продолжает в нормальный режим запуска. При будущих запусках он запросит парольную фразу хранилища для разблокировки кошелька.

Для автоматического запуска без ручного ввода установите переменную окружения `LASERSELL_WALLET_PASSPHRASE`.

## Устранение неполадок

**«config file not found; run --setup in an interactive terminal»**
Мастер требует реальный TTY. Подключитесь по SSH к машине или откройте локальный терминал и запустите `lasersell --setup`.

**Импортированная сид-фраза даёт неправильный адрес**
LaserSell использует стандартный путь деривации Solana (`m/44'/501'/0'/0'`). Если ваш кошелёк использует другую деривацию, импортируйте через base58 приватный ключ или keypair JSON.
