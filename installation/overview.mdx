LaserSell is distributed as a **precompiled, closed-source binary**. There is no build-from-source path.

### Supported Operating Systems

- **Linux** (x86_64, arm64 / Raspberry Pi)
- **macOS** (Intel & Apple Silicon)
- **Windows** (x86_64)
- **Docker** (advanced / optional)

### Installation Methods

- **Native install (recommended)**\
  Best performance, full OS integration.
- **Docker install (advanced)**\
  For containerized or server-based environments.

> All official binaries and install scripts are hosted at\
> [**https://dl.lasersell.io**](https://dl.lasersell.io)

### After Installation

Once installed, continue to **Quickstart** to:

- Run initial setup
- Unlock your wallet
- Verify your first live session
