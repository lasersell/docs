---
title: "Overview"
description: "This section covers how to install LaserSell on all supported platforms."
---

LaserSell is distributed as a **proprietary, precompiled binary**. The official install entrypoint for all platforms is:

```shellscript
curl -fsSL https://dl.lasersell.io/install.sh | bash
```

This install script is the **recommended** method and is used for Linux, macOS, and Windows (via WSL).

### Supported Platforms

- **Linux (Debian/Ubuntu)** via `install.sh` or manual APT setup
- **macOS** via `install.sh` or Homebrew (or tarball mode)
- **Windows** is supported **via WSL** (not a native `.exe` install)
- **Docker** (optional / advanced) using the official `lasersell/lasersell` image

### Migrating from the old repo

If you previously used the old `lasersell.github.io/apt` repo, run:

```shellscript
curl -fsSL https://dl.lasersell.io/migrate.sh | bash
```

### Upgrading

- APT: `sudo apt update && sudo apt install lasersell`
- Homebrew: `brew upgrade lasersell`
- Tarball install: rerun `install.sh` (overwrites the binary)

### CLI flags

| Flag | Short | Description |
| :--- | :--- | :--- |
| `--setup` | | Force-run the onboarding wizard (even if a config already exists) |
| `--config <path>` | `-f` | Path to a custom config file (required for `--no-tui` and `--verbose` modes) |
| `--verbose` | `-v` | Disable the TUI and output plain log lines instead (headless mode) |
| `--no-tui` | | Disable the TUI (requires `-f`). Equivalent to `--verbose` |
| `--smoke` | | Run a connectivity health check (RPC + exit API) and exit. Prints `SMOKE OK` or `SMOKE FAIL <step>` |
| `--export-private-key` | | Export your wallet's base58 private key to stdout. Optionally accepts a keystore path after `=` |

Notes:

- `--smoke`, `--setup`, and `--export-private-key` are mutually exclusive.
- `--no-tui` and `--verbose` both require `-f <path>` to specify the config file.
- In non-TUI mode, the keystore passphrase is read from the `LASERSELL_WALLET_PASSPHRASE` env var, falling back to an interactive stdin prompt.
