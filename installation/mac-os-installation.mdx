---
title: "macOS Installation"
description: "LaserSell supports both Intel and Apple Silicon (M-series) Macs."
---

### Recommended Method: Homebrew

LaserSell provides an official Homebrew tap.

```shellscript
brew tap lasersell/lasersell
brew install lasersell
```

Homebrew will automatically install the correct binary for your architecture.

### Verify Installation

```shellscript
lasersell --version
```
