---
title: "macOS Installation"
description: "LaserSell supports both Intel and Apple Silicon (M-series) Macs."
---

### macOS via install.sh (recommended)

```shellscript
curl -fsSL https://dl.lasersell.io/install.sh | bash
```

### macOS via Homebrew

```shellscript
brew tap lasersell/lasersell
brew install lasersell
```

### macOS without Homebrew (tarball mode)

If you want a tarball-based install path:

```shellscript
curl -fsSL https://dl.lasersell.io/install.sh | bash -s -- --method tar
```

### Verify

```shellscript
lasersell --version
```
