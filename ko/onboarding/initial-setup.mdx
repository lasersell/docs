title: "초기 설정"
description: "구성과 암호화된 지갑을 생성하는 LaserSell 설정 마법사의 안내."
---

설정 마법사는 첫 실행 시 자동으로 실행됩니다. `lasersell --setup`으로 언제든지 다시 실행할 수 있습니다.

## 필요한 것

- [Helius](https://helius.dev), [Chainstack](https://chainstack.com) 또는 Solana RPC 제공자의 **RPC URL**
- [app.lasersell.io](https://app.lasersell.io)의 **LaserSell API 키** (무료)
- 지갑 계획: 새 지갑 생성 또는 **시드 구문**, **base58 비밀키**, **Solana 키페어 JSON** 준비

## 마법사가 생성하는 것

- `~/.lasersell/config.yml` (RPC, API 키, 전략)
- `~/.lasersell/wallet.keystore.json` (암호화된 지갑)

## 마법사 안내

### 자격 증명

```
RPC URL (private recommended): https://your-private-rpc.example.com
LaserSell API key: ********
```

API 키 입력은 마스크됩니다. 프로덕션에는 전용 RPC를 강력히 권장합니다.

### 전략

마법사가 각 청산 조건을 구성하라는 프롬프트를 표시합니다. 거래 스타일과 위험 허용 범위에 맞는 값을 설정하세요.

- **목표 수익**: 수익이 이 %에 도달하면 매도 (예: `6%`)
- **손절매**: 손실이 이 %에 도달하면 매도 (예: `10%`)
- **트레일링 스탑**: 수익이 최고점에서 이 %만큼 하락하면 매도 (예: `5%`)
- **데드라인 타임아웃**: N초 후 강제 매도 (예: `120`)
- **슬리피지 허용**: 매도에 대한 최대 슬리피지 (예: `20%`)
- **졸업 시 매도**: 토큰이 AMM으로 마이그레이션 시 자동 매도 (`y/N`)

최소 하나의 청산 조건(목표 수익, 손절매, 트레일링 스탑 또는 데드라인)이 활성화되어야 합니다. `~/.lasersell/config.yml`을 편집하고 데몬을 재시작하거나 `lasersell --setup`을 재실행하여 나중에 변경할 수 있습니다.

### 지갑

**기존 키스토어 감지?** 재사용하거나 다른 지갑을 생성/가져올 수 있습니다.

**새 지갑 생성:** LaserSell이 12단어 시드 구문을 생성하고 한 번만 표시합니다. 즉시 저장하세요.

<Warning>
시드 구문은 다시 표시되지 않습니다. 잃으면 지갑을 잃게 됩니다.
</Warning>

**기존 지갑 가져오기:** 다음 중 하나를 선택합니다:

1. **Base58 비밀키** (입력은 마스크됨)
2. **시드 구문** (유도 경로 `m/44'/501'/0'/0'` 사용)
3. **Solana 키페어 JSON** 파일 경로

가져오기 후 표시된 공개키가 의도한 지갑과 일치하는지 확인하세요.

### 패스프레이즈

키스토어를 암호화할 패스프레이즈를 설정합니다:

```
Keystore passphrase: ********
Confirm passphrase: ********
```

<Warning>
"비밀번호 찾기"가 없습니다. 패스프레이즈를 잃으면 키스토어를 복구할 수 없습니다. 복구 경로는 시드 구문 또는 개인키에서 지갑을 다시 가져오는 것입니다.
</Warning>

### 검토 및 저장

LaserSell이 설정 요약을 보여주고 구성 저장을 요청합니다. 확인하여 저장합니다.

## 설정 후

LaserSell이 정상 시작으로 계속됩니다. 향후 실행 시 지갑 잠금 해제를 위해 키스토어 패스프레이즈를 요청합니다.

헤드리스/자동화된 실행을 위해 `LASERSELL_WALLET_PASSPHRASE` 환경 변수를 설정하세요.

## 문제 해결

**"config file not found; run --setup in an interactive terminal"**
마법사에는 실제 TTY가 필요합니다. 머신에 SSH로 접속하거나 로컬 터미널을 열고 `lasersell --setup`을 실행하세요.

**가져온 시드 구문이 잘못된 주소를 제공함**
LaserSell은 Solana의 표준 유도 경로(`m/44'/501'/0'/0'`)를 사용합니다. 지갑이 다른 유도를 사용하면 base58 비밀키 또는 키페어 JSON으로 가져오세요.
