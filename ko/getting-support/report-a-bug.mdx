## 보안 취약점

LaserSell 제품에서 보안 취약점을 발견하면 책임감 있게 신고해 주세요:

- **이메일:** [security@lasersell.io](mailto:security@lasersell.io)
- **PGP:** [PGP 키](https://www.lasersell.io/lasersell-pgp-key.asc)로 보고서를 암호화하는 것을 권장합니다. 전체 공개 지침은 [www.lasersell.io/.well-known/security.txt](https://www.lasersell.io/.well-known/security.txt)에 있습니다.

보안 취약점에 대해 공개 GitHub 이슈를 열지 **마세요**.

## 소프트웨어 버그

LaserSell 또는 LaserSell SDK의 버그는 다음 채널을 사용하세요:

- **GitHub Issues:**
  - [lasersell/lasersell](https://github.com/lasersell/lasersell/issues) (클라이언트)
  - [lasersell/lasersell-sdk](https://github.com/lasersell/lasersell-sdk/issues) (SDK)
- **Discord 티켓:** [discord.gg/lasersell](https://discord.gg/lasersell)에서 지원 티켓을 열어주세요

신고 시 가능한 한 많은 세부사항을 포함해 주세요: 재현 단계, 오류 메시지, OS/플랫폼, 버전 (`lasersell --version`).

### 해결 프로세스

확인된 버그를 신속하게 분류하고 수정하는 것을 목표로 합니다.

- **알림:** 수정이 구현되면 최신 업데이트를 받을 수 있도록 알려드립니다.
- **업데이트:** [LaserSell 공지](https://t.me/lasersell) Telegram 채널 또는 Discord의 `#announcements` 채널에서 수정이 빌드에 포함되어 있는지 확인하세요.
