제품이 Solana 토큰 거래와 관련이 있다면, LaserSell이 그 뒤의 실행 레이어를 구동할 수 있습니다.

LaserSell 클라이언트에서 실행되는 동일한 Exit Intelligence 엔진이 [LaserSell SDK 및 API](/api/overview)를 통해 이용 가능합니다. Telegram 봇, 웹 플랫폼, 모바일 앱, 브라우저 확장, CLI 도구, 포트폴리오 관리자, 런치패드 서비스, 카피 트레이딩 시스템, 시그널 그룹, 분석 대시보드. 사용자가 Solana에서 토큰을 매수하거나 매도한다면, LaserSell은 실시간 전략 평가, 멀티 프로토콜 트랜잭션 빌딩, 내장 수익화 기능을 갖춘 인프라를 제공합니다.

청산 엔진을 직접 구축할 필요가 없습니다. AMM 상태를 추적하거나 프로토콜별 인스트럭션을 처리할 필요가 없습니다. 한 번 통합하면 모든 주요 Solana 런치패드와 DEX를 지원합니다.

## 실시간 Exit Intelligence

Exit Intelligence Stream은 지갑을 감시하고, 포지션을 추적하고, 실시간으로 전략을 평가하며, 임계값이 충족되는 순간 사전 구축된 청산 트랜잭션을 전달하는 지속적인 WebSocket 연결입니다.

자체적으로 손익 추적, 가격 모니터링, 청산 로직을 처음부터 구축하는 대신, 스트림에 연결하면 사전 구축된 트랜잭션을 받게 됩니다. 애플리케이션이 전략(목표 수익, 손절매, 트레일링 스탑, 데드라인, 졸업 시 매도)을 정의하면 LaserSell이 나머지를 처리합니다.

이는 실행 엔진을 직접 구축하거나 유지 관리하지 않고도 사용자에게 자동 청산 기능을 제공할 수 있음을 의미합니다.

## 매수 및 매도 트랜잭션 빌더

REST API는 `/v1/buy` 및 `/v1/sell`을 통해 요청 시 서명되지 않은 스왑 트랜잭션을 빌드합니다. Pump.fun, PumpSwap, Meteora DBC, Meteora DAMM V2, Raydium Launchpad, Raydium CPMM 전반에 걸친 완전한 프로토콜 지원을 제공합니다.

AMM 상태를 추적하거나, 경로를 계산하거나, 프로토콜별 인스트럭션 형식을 처리할 필요가 없습니다. 민트 주소와 금액으로 요청을 보내면, API가 서명하고 제출할 준비가 된 최적화된 서명되지 않은 트랜잭션을 반환합니다.

## 사용자의 키, 사용자의 서명

API가 반환하는 모든 트랜잭션은 서명되지 않은 상태입니다. 아키텍처에 맞게 서명을 처리합니다. 키가 LaserSell 인프라에 전달되는 일은 없습니다.

애플리케이션이 사용자를 대신하여 지갑을 관리하든, 사용자가 직접 서명하든, LaserSell은 서명 처리 방식에 제약 없이 기존 키 관리 설정에 적합합니다.

## 내장 수익화

사용자가 LaserSell을 통해 실행하는 모든 트랜잭션에 파트너 수수료를 포함하여 직접 지갑으로 라우팅할 수 있습니다. 각 요청에 수수료(베이시스 포인트 또는 고정 lamport)를 설정하면 스왑의 일부로 온체인에서 원자적으로 적용됩니다.

인보이스도, 정산도, 오프체인 결제 흐름도 필요 없습니다. 수익은 거래와 동일한 트랜잭션에서 지갑에 도착합니다.

구현 세부사항은 [파트너 수수료](/api/reference/partner-fees)를 참조하세요. 엔터프라이즈 파트너는 맞춤 수익화 방식에 대해 저희와 협의할 수 있습니다.

## 멀티 프로토콜 지원

단일 통합으로 주요 Solana 거래 프로토콜 전반에 걸친 지원을 받을 수 있습니다:

- **Pump.fun** 본딩 커브 및 **PumpSwap** 마켓
- **Meteora** Dynamic Bonding Curve (DBC) 및 Dynamic AMM V2
- **Raydium** Launchpad 및 CPMM 풀

Solana에서 새로운 프로토콜이 출시되면, LaserSell이 지원을 추가하고 통합이 자동으로 적용됩니다. 사용자 측에서 코드 변경이 필요 없습니다.

## LaserSell이 적합한 곳

사용자가 Solana 토큰을 보유하고 매도해야 하는 모든 제품이 LaserSell의 혜택을 받을 수 있습니다. 아래는 몇 가지 예시이지만 전체 목록은 아닙니다. 제품이 Solana에서 토큰 거래와 관련이 있다면, Exit Intelligence가 적합한 곳이 있을 것입니다.

### 봇 (Telegram, Discord, CLI 또는 커스텀)

봇 사용자에게 모든 거래에 자동 TP, SL, 트레일링 스탑을 제공하세요. 사용자 지갑별로 Exit Intelligence Stream에 연결하고, 조건이 충족되면 청산 알림을 전달하거나 자동으로 매도를 실행하세요. 파트너 수수료로 봇이 처리하는 모든 거래를 수익화할 수 있습니다.

### 웹 및 모바일 거래 플랫폼

거래 UI에 원클릭 매수/매도 및 자동 청산 전략을 추가하세요. 트랜잭션 빌더가 프로토콜 라우팅을 처리하고, Exit Intelligence Stream이 라이브 대시보드를 구동하는 실시간 손익 업데이트를 제공합니다.

### 런치패드 및 배포자 도구

토큰 크리에이터를 위한 출시 후 청산 워크플로우를 자동화하세요. 배포자가 도구를 통해 출시하면, LaserSell이 포지션을 모니터링하고 크리에이터의 청산 계획을 자동으로 실행할 수 있습니다.

### 카피 트레이딩 및 시그널 그룹

시그널이 발생하거나 리더가 포지션에 진입하면, LaserSell이 모든 팔로워의 청산을 자동으로 관리할 수 있습니다. 각 지갑은 수동 개입 없이 자체 전략 평가와 청산 실행을 받습니다.

### 포트폴리오 트래커, 분석 및 알림

Exit Intelligence Stream에 연결하여 지원되는 모든 프로토콜에서 실시간 포지션 추적, 손익 업데이트, 청산 이벤트 데이터를 받으세요. 분석 대시보드, 알림 시스템, 위험 관리 도구를 구동하세요.

### 브라우저 확장 및 지갑 애드온

브라우징 경험에 직접 자동 청산 기능을 내장하세요. 사용자가 어떤 플랫폼에서든 매수하면 확장이 LaserSell을 사용하여 백그라운드에서 매도를 관리합니다.

### 그 외 모든 것

사용자가 Solana에서 토큰을 거래하고 있다면, LaserSell의 청산 엔진과 트랜잭션 빌더는 플러그인 방식으로 설계되었습니다. API는 사용자 관점에서 프로토콜에 구애받지 않습니다. 민트 주소와 금액을 보내면 최적화된 트랜잭션을 받습니다. Exit Intelligence Stream은 지갑을 감시하고 실시간으로 청산 신호를 전달합니다. 이를 사용자에게 어떻게 표시할지는 전적으로 사용자에게 달려 있습니다.
