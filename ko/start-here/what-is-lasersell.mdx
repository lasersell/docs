LaserSell은 Solana 네트워크를 위한 자동 청산 엔진입니다. 온체인에서 지갑을 모니터링하고, 새로 매수한 토큰을 감지하며, 실시간으로 손익을 추적하고, 설정한 조건이 충족되면 자동으로 매도합니다. 키는 항상 사용자가 관리합니다.

## 작동 방식

1. **토큰을 매수합니다.** 이미 사용 중인 어떤 플랫폼에서든 매수합니다.
2. **LaserSell이 매수를 감지합니다.** 온체인에서 지갑을 모니터링합니다.
3. **청산 전략이 자동으로 실행됩니다.** LaserSell은 모든 가격 업데이트마다 목표 수익, 손절매, 트레일링 스탑, 데드라인, 졸업 시 매도 조건을 평가합니다.
4. **조건이 충족되는 순간 LaserSell이 매도합니다.** 트랜잭션이 생성되고, 서명된 후, Solana 네트워크에 직접 전송됩니다.

수동 클릭도, 확인 팝업도, 탭 전환도 필요 없습니다.

## LaserSell이 아닌 것

- **수탁 서비스가 아닙니다.** 저희는 개인키에 접근할 수 없습니다.
  - **데스크톱 앱:** 키는 기기에서 암호화되어 로컬에 저장됩니다. 모든 트랜잭션에 사용자가 직접 서명합니다.
  - **웹 앱:** 키는 [Privy의 Trusted Execution Environment(TEE)](https://docs.privy.io/guide/security), 즉 LaserSell과 Privy 직원 모두 접근할 수 없는 하드웨어 격리 보안 영역에 저장됩니다. Privy는 [Stripe 계열사](https://privy.io/blog/announcing-our-acquisition-by-stripe)입니다. 언제든지 키를 내보낼 수 있습니다.
- **투자 조언이 아닙니다.** LaserSell은 실행 인프라입니다. 사용자가 전략을 설정하고 LaserSell이 실행합니다.

## 거래 방법 선택

LaserSell은 사용자의 환경에 맞춰 제공됩니다. 모든 클라이언트가 동일한 모니터링 인프라에 연결되고 동일한 전략을 실행합니다.

### 웹 앱

[LaserSell 웹 앱](/web-app/overview)은 시작하는 가장 쉬운 방법입니다. 로그인하고, 지갑에 자금을 입금하고, 전략을 설정하면, 브라우저를 닫아도 LaserSell이 연중무휴 24시간 거래를 실행합니다.

- 다운로드 불필요, 설치 불필요, 모든 기기에서 사용 가능
- Privy의 Trusted Execution Environment를 통한 자동 실행
- 실시간 손익 업데이트와 함께 포지션 모니터링
- 다른 지갑의 거래를 자동으로 미러링하는 카피 트레이딩
- 부재 중 거래 이벤트에 대한 푸시 알림

[웹 앱 시작하기 →](/web-app/overview)

### 데스크톱 앱

[LaserSell Desktop 앱](/desktop/overview)은 완전한 로컬 제어를 원하는 트레이더를 위해 제작되었습니다. 키는 기기에 보관되고, 모든 트랜잭션에 사용자가 직접 서명하며, RPC와 트랜잭션 전송 방식을 직접 선택할 수 있습니다.

- 실시간 포지션 모니터링이 있는 네이티브 macOS 애플리케이션
- 다단계 청산 전략과 미러 트레이딩
- 스왑 기능이 포함된 내장 Solana 지갑
- 일일 손익 차트, 업적, 인사이트가 포함된 분석
- Touch ID 잠금 해제 및 7개 언어 지원

[Desktop 앱 시작하기 →](/desktop/installation)

### API 및 SDK

자체 클라이언트, 봇 또는 거래 플랫폼을 구축하려는 개발자를 위해 동일한 청산 엔진을 [LaserSell API 및 SDK](/api/overview)를 통해 이용할 수 있습니다.

- [TypeScript](/api/sdk/typescript), [Python](/api/sdk/python), [Rust](/api/sdk/rust), [Go](/api/sdk/go)로 이용 가능
- 트랜잭션 빌딩 API 및 실시간 스트림에 대한 전체 접근
- 맞춤 전략, 알림, 대시보드 또는 자율 에이전트 구축

[API 살펴보기 →](/api/overview)

## LaserSell 사용자

### 트레이더

Solana에서 밈코인을 거래한다면, LaserSell은 모든 포지션을 수동으로 감시할 필요 없이 자동 청산 기능을 제공합니다. 전략을 한 번 설정하고 실행하세요.

- 목표 수익, 손절매, 트레일링 스탑, 데드라인 타임아웃, 졸업 시 매도가 모두 병렬로 실행됩니다.
- 모든 진입 방법과 호환됩니다. 이미 사용 중인 어떤 플랫폼이나 지갑에서든 매수하세요.

### 배포자

RapidLaunch, Uxento, J7 Tracker 같은 도구로 토큰을 출시한다면, LaserSell은 개인 청산 계획을 자동화하여 출시와 커뮤니티에 집중할 수 있게 합니다.

### 개발자

LaserSell의 실시간 청산 인텔리전스, 멀티 프로토콜 지원, 트랜잭션 빌딩 위에 자신만의 트레이딩 봇, 플랫폼 또는 맞춤 전략을 구축하세요.

## 지원 프로토콜

LaserSell은 주요 Solana 런치패드와 DEX를 지원합니다:

- Pump.fun 및 PumpSwap
- Meteora DBC 및 DAMM V2
- Raydium Launchpad 및 CPMM
