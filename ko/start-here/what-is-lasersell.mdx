LaserSell은 Solana 네트워크를 위한 자동 청산 엔진입니다. 백그라운드에서 실행되어 새로 매수한(또는 생성한) 토큰을 감지하고, 실시간으로 손익을 모니터링하며, 설정한 조건이 충족되면 자동으로 매도합니다. 키는 암호화된 상태로 사용자의 기기에 안전하게 보관됩니다.

## 작동 방식

1. **토큰을 매수합니다.** 이미 사용 중인 어떤 플랫폼에서든 매수합니다.
2. **LaserSell이 매수를 감지합니다.** 온체인에서 지갑을 모니터링합니다.
3. **청산 전략이 자동으로 실행됩니다.** LaserSell은 모든 가격 업데이트마다 목표 수익, 손절매, 트레일링 스탑, 데드라인, 졸업 시 매도 조건을 평가합니다.
4. **조건이 충족되는 순간 LaserSell이 매도합니다.** 트랜잭션이 생성되고, 로컬에서 서명된 후, Solana 네트워크에 직접 전송됩니다.

수동 클릭도, 확인 팝업도, 탭 전환도 필요 없습니다.

## LaserSell이 아닌 것

- **수탁 서비스가 아닙니다.** 개인키는 사용자의 기기에서 암호화됩니다. 저희는 절대 접근할 수 없습니다.
- **지갑을 연결하는 웹사이트가 아닙니다.** LaserSell은 사용자의 기기에서 실행되는 로컬 클라이언트이거나, 사용자의 애플리케이션에 통합하는 SDK/API입니다.

## 데스크톱 앱

[LaserSell Desktop 앱](/desktop/overview)은 LaserSell을 사용하는 대표적인 방법입니다. 실시간 포지션 모니터링, 다단계 청산 전략, 미러 트레이딩(모든 지갑에서 카피 트레이딩), 내장 Solana 지갑(스왑 포함), 일일 손익 차트가 포함된 분석, 업적, Touch ID 잠금 해제, 7개 언어 지원이 있는 네이티브 macOS 애플리케이션입니다.

## LaserSell 사용자

### 트레이더

Solana에서 밈코인을 거래한다면, LaserSell은 모든 포지션을 수동으로 감시할 필요 없이 자동 청산 기능을 제공합니다. 전략을 한 번 설정하고 실행하세요.

- 목표 수익, 손절매, 트레일링 스탑, 데드라인 타임아웃, 졸업 시 매도가 모두 병렬로 실행됩니다.
- 모든 진입 방법과 호환됩니다. 이미 사용 중인 어떤 플랫폼이나 지갑에서든 매수하세요.
- 시각적 인터페이스를 원하면 [Desktop 앱](/desktop/overview)을, 헤드리스 운영을 원하면 [CLI](/start-here/quickstart)를 사용하세요.

### 배포자

RapidLaunch, Uxento, J7 Tracker 같은 도구로 토큰을 출시한다면, LaserSell은 개인 청산 계획을 자동화하여 출시와 커뮤니티에 집중할 수 있게 합니다.

### 개발자

LaserSell 클라이언트를 구동하는 동일한 청산 엔진이 [LaserSell SDK 및 API](/api/overview)를 통해 이용 가능합니다. LaserSell의 실시간 청산 인텔리전스, 멀티 프로토콜 지원, 트랜잭션 빌딩 위에 자신만의 트레이딩 봇, 플랫폼 또는 맞춤 전략을 구축하세요.

[TypeScript](/api/sdk/typescript), [Python](/api/sdk/python), [Rust](/api/sdk/rust), [Go](/api/sdk/go)로 이용 가능합니다.
