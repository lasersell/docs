## 연결 및 계정

- **연결 상태** — 앱이 LaserSell 스트림에 연결되어 있는지 표시 (녹색 = 연결됨, 빨간색 = 연결 끊김).
- **계정 티어** — 현재 구독: Personal (무료), Professional 또는 Advanced.
- **모니터링 일시 중지** — 연결을 끊지 않고 모든 스트림 모니터링과 자동 매도를 임시로 중지합니다. 포지션을 수동으로 보유하려 할 때 유용합니다.

## 네트워크

### API 키

LaserSell API 키입니다. 기본적으로 마스크되며 표시/숨기기 토글과 복사 버튼이 있습니다. 새 키가 필요하면 로그인 흐름을 사용하여 생성하세요.

### RPC URL

Solana RPC 엔드포인트입니다. Helius, QuickNode, Triton 같은 제공자의 전용 RPC가 있다면 여기에 설정하여 더 빠르고 안정적인 트랜잭션을 받으세요. 기본값은 공용 엔드포인트입니다.

### 전송 모드

LaserSell이 네트워크에 트랜잭션을 제출하는 방식:

- **RPC** (기본값) — RPC 엔드포인트를 통한 직접 제출. 추가 수수료 없음.
- **Helius Sender** — 더 나은 랜딩률을 위해 Helius를 통한 최적화된 트랜잭션 전달. Helius API 키와 빌더 팁(권장: 0.001 SOL) 필요.
- **Astralane Iris** — MEV 보호 트랜잭션 라우팅. Astralane API 키와 지역 선택(프랑크푸르트, 암스테르담, 뉴욕, 샌프란시스코, 도쿄, 싱가포르 등) 필요.

## 디스플레이

- **통화** — 포트폴리오 가치와 손익을 위한 표시 통화 선택. 아메리카, 유럽, 아시아 태평양, 중동의 20개 통화 지원.
- **USD1 표시** — 사이드바에 USD1 스테이블코인 잔액 표시.
- **미러 USD1 거래** — USD1 표시 카피 트레이딩 활성화.
- **메뉴 바 아이콘** — 빠른 접근을 위해 macOS 메뉴 바에 LaserSell 표시.
- **알림 소리** — 거래 실행 시 코인 사운드 재생.
- **시작 시 연결** — 앱 실행 시 자동으로 스트림에 연결. 스트림 모니터링 없이 독립형 지갑으로 앱을 사용하려면 비활성화하세요.

## 거래

- **빠른 매수 금액 (SOL)** — 빠른 매수 버튼의 사전 설정 SOL 금액 (예: 0.1, 0.5, 1.0 SOL).
- **빠른 매수 금액 (USD1)** — 사전 설정 USD1 금액 (USD1 매수 활성화 시 표시).
- **빠른 매도 비율** — 부분 매도를 위한 사전 설정 매도 비율 (예: 10%, 25%, 50%, 100%).
- **슬리피지 허용 범위** — 거래에 허용되는 최대 가격 슬리피지(베이시스 포인트).

## 보안

Touch ID, 자동 잠금, 암호화에 대한 자세한 내용은 [보안](/desktop/security)을 참조하세요.

## 언어 및 다국어 지원

LaserSell은 전 세계 트레이더를 위해 설계되었습니다. 전체 앱 인터페이스가 7개 언어로 완전히 현지화되어 있으며, 레이아웃은 각 언어에 맞게 자동으로 조정됩니다.

<video autoPlay muted loop playsInline src="/videos/LaserSell International.mp4" />

**지원 언어:**

- English
- Chinese (中文)
- Japanese (日本語)
- Korean (한국어)
- Spanish (Español)
- Russian (Русский)
- Arabic (العربية)

언어 전환은 즉시 이루어집니다. 모든 화면, 모든 레이블, 모든 알림이 재시작 없이 즉시 업데이트됩니다.

### 오른쪽에서 왼쪽(RTL) 지원

LaserSell은 Arabic을 위한 오른쪽에서 왼쪽(RTL) 레이아웃을 완벽하게 지원합니다. Arabic으로 전환하면 전체 인터페이스가 반전됩니다. 내비게이션이 오른쪽으로 이동하고, 텍스트가 오른쪽에서 왼쪽으로 정렬되며, 모든 컴포넌트가 자연스럽게 재배치됩니다. 이것은 단순한 텍스트 교체가 아닙니다. 사이드바, 포지션 목록, 전략 패널, 지갑 뷰, 분석 차트를 포함한 전체 레이아웃이 RTL 읽기 순서에 맞게 재구성됩니다. Arabic을 사용하는 트레이더는 다른 모든 언어와 동일한 세련되고 네이티브한 경험을 얻을 수 있습니다.

### 다중 통화 표시

포트폴리오 가치, 손익, 거래 금액을 20개 통화 중 원하는 통화로 표시할 수 있습니다:

- **아메리카** — USD, CAD
- **유럽** — EUR, GBP, CHF
- **아시아 태평양** — JPY, CNY, KRW, INR, SGD, HKD, AUD, NZD
- **중동 및 아프리카** — AED, SAR, KWD, BHD, QAR, OMR, EGP

모든 값은 환율이 변동됨에 따라 실시간으로 업데이트됩니다. 자국 통화를 선택하여 본인에게 의미 있는 숫자로 성과를 확인하세요.

## 개인정보

- **크래시 리포트** — Sentry를 통한 익명 오류 리포팅 토글. 지갑 데이터나 거래 정보는 포함되지 않습니다.

## 정보

- **버전** — 현재 앱 버전.
- **플랫폼** — 운영 체제.
- **도움말** — 지원 리소스 링크.
- **오류 리포트 복사** — 지원팀과 공유하기 위한 최근 오류 형식화.
- **로그아웃** — API 키를 지우고 로그인 화면으로 돌아갑니다.
