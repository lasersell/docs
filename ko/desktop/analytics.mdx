분석 페이지는 거래 성과에 대한 데이터 기반 뷰와 마일스톤을 추적하는 게이미피케이션 업적 시스템을 제공합니다.

> **참고:** 모든 분석, 인사이트 및 업적은 LaserSell을 통한 거래 활동만을 기반으로 합니다. 다른 도구를 통하거나 온체인에서 직접 실행한 거래는 포함되지 않습니다.

## 개요

기간(24시간, 7일, 30일 또는 전체 기간)을 선택하여 주요 지표를 확인하세요:

- **총 P&L** — 표시 통화 기준 순 수익 또는 손실
- **총 거래량** — 견적 단위 기준 총 거래량
- **승률** — 수익으로 마감한 거래 비율
- **총 거래 수** — 청산된 포지션 수
- **평균 P&L** — 거래당 평균 수익 또는 손실
- **평균 보유 시간** — 포지션이 열려 있던 평균 시간
- **최고 거래** — 단일 거래 최고 수익
- **최악 거래** — 단일 거래 최대 손실

### 일일 손익 차트

지난 90일간의 일일 수익과 손실을 보여주는 막대 차트입니다. 녹색 막대는 수익일, 빨간 막대는 손실일입니다. 막대 위에 마우스를 올리면 정확한 수치를 볼 수 있습니다.

### 청산 사유 분석

청산을 트리거한 이유의 시각적 분석:

- **목표 수익** — 전략이 목표 수익 임계값에 도달
- **트레일링 스탑** — 가격이 최고점에서 트레일링 비율만큼 하락
- **손절매** — 가격이 손절매 임계값에 도달
- **졸업** — 토큰이 본딩 커브에서 DEX로 마이그레이션
- **데드라인** — 포지션 타임아웃

어떤 청산 조건이 가장 많이 작동하는지, 전략을 조정할 필요가 있는지 이해하는 데 도움이 됩니다.

## 인사이트 (Tier 1+)

최근 거래 패턴에 기반한 맞춤형 제안. 각 인사이트에는 심각도 수준(높음, 중간, 낮음)과 구체적인 권장 사항이 있습니다.

인사이트는 거래 히스토리에서 서버 측에서 생성되며 주기적으로 업데이트됩니다. Professional 티어 이상으로 제한됩니다.

무료 티어인 경우 업그레이드 프롬프트가 표시됩니다.

## 업적

**10개 카테고리**에서 Common부터 Legendary까지 희귀도 티어로 진행 상황을 추적하세요.

### 카테고리

| 카테고리 | 추적 대상 |
| :--- | :--- |
| 거래량 | 거래량 마일스톤 |
| 거래 수 | 완료된 거래 수 |
| 연속 기록 | 연속 수익 거래일 |
| 수익 | 누적 수익 목표 |
| 속도 | 빠른 실행 시간 |
| 단일 거래 | 단일 거래 손익 기록 |
| 승률 | 승률 비율 마일스톤 |
| 미러 | 카피 트레이딩 마일스톤 |
| 시크릿 | 숨겨진 업적 (거래하며 발견) |
| 마스터리 | 고급 조합 업적 |

### 희귀도 티어

- **Common** — 입문 수준 마일스톤
- **Uncommon** — 중간 수준의 성취
- **Rare** — 중요한 업적
- **Epic** — 탁월한 거래 마일스톤
- **Legendary** — 시스템에서 가장 희귀한 업적

### 연속 기록

현재 및 최장 연속 기록은 업적 탭 상단에 표시됩니다. LaserSell 프로토콜을 통해 라우팅된 수익 거래가 최소 한 건 있으면 연속 기록일이 카운트됩니다.

### 업적 작동 방식

업적은 서버에서 매시간 확인됩니다. LaserSell 프로토콜을 통해 라우팅된 거래만 업적 진행에 반영됩니다. 새로운 업적을 잠금 해제하면 앱이 알림을 보냅니다.

<Note>
잠금 해제된 업적은 향후 보상으로 교환할 수 있게 될 것입니다. LaserSell 토큰은 없으며 에어드롭도 없을 것입니다.
</Note>
