LaserSell Desktop을 처음 열면 온보딩 마법사가 다섯 단계를 안내합니다. 전체 과정은 약 2분이 소요됩니다.

## 단계 1: 로그인

LaserSell 계정으로 로그인하여 API 키를 받으세요. 두 가지 옵션이 있습니다:

- **브라우저 로그인:** "Sign In"을 클릭하고 브라우저에서 인증하세요. 앱이 딥 링크를 통해 자동으로 API 키를 받습니다.
- **수동 붙여넣기:** 이미 API 키가 있다면 직접 붙여넣으세요.

아직 계정이 없다면 "Sign Up"을 클릭하여 생성하세요. 계정은 무료이며 즉시 생성됩니다.

## 단계 2: 지갑 설정

LaserSell이 모니터링할 Solana 지갑이 최소 하나 필요합니다. 하나를 선택하세요:

- **기존 지갑 가져오기.** Base58 개인키를 붙여넣으세요. 이미 자금이 있는 거래 지갑이 있는 경우 유용합니다.
- **새 지갑 생성.** LaserSell이 로컬에서 새 키페어를 생성합니다. 거래 전에 SOL로 자금을 넣어야 합니다.

어느 쪽이든 암호화 패스프레이즈(8자 이상)를 설정합니다. 이 패스프레이즈는 XChaCha20Poly1305를 사용하여 디스크에서 개인키를 암호화합니다. LaserSell은 키를 평문으로 저장하거나 전송하지 않습니다.

<Tip>
나중에 [지갑 > 관리](/desktop/wallet) 탭에서 더 많은 지갑을 추가할 수 있습니다. 계정 티어에 따라 동시에 사용할 수 있는 지갑 수가 결정됩니다.
</Tip>

## 단계 3: 초기 전략

시작 청산 전략을 설정하세요:

- **목표 수익 %** — LaserSell이 자동 매도하는 수익 임계값 (기본값: 25%).
- **손절매 %** — LaserSell이 포지션을 청산하는 손실 임계값 (기본값: 15%).

이것은 시작 값입니다. 온보딩 후 [청산 전략](/desktop/exit-strategy) 페이지에서 트레일링 스탑, 데드라인, 졸업 시 매도, 청산 래더 등을 구성할 수 있습니다.

## 단계 4: RPC URL

선택적으로 커스텀 Solana RPC 엔드포인트를 설정하세요. Helius, QuickNode, Triton 또는 다른 제공자의 전용 RPC가 있다면 여기에 URL을 붙여넣어 더 빠르고 안정적인 트랜잭션 전달을 받으세요.

이 단계를 건너뛰면 앱은 기본 공용 엔드포인트를 사용합니다. 나중에 [설정 > 연결](/desktop/settings)에서 변경할 수 있습니다.

## 단계 5: 크래시 리포팅

익명 크래시 리포트를 보낼지 선택하세요. 이는 버그를 찾고 수정하는 데 도움이 됩니다. 지갑 데이터, 키, 거래 정보는 절대 포함되지 않습니다. 설정에서 언제든지 변경할 수 있습니다.

## 잠금 해제

온보딩 후, 이후 실행 시 잠금 해제 화면이 표시됩니다. 지갑 패스프레이즈를 입력하여 키를 복호화하고 모니터링을 시작하세요.

패스프레이즈가 다른 여러 지갑이 있는 경우, 개별적으로 잠금 해제하거나 이 세션에 필요하지 않은 지갑을 건너뛸 수 있습니다.

macOS에서는 [설정 > 보안](/desktop/security)에서 **Touch ID**를 활성화하여 더 빠른 잠금 해제가 가능합니다.

## 다음은 무엇인가

잠금 해제 후 앱이 LaserSell Exit Intelligence 스트림에 연결하고 지갑 모니터링을 시작합니다. 보유 중인(또는 이 시점부터 매수하는) 모든 토큰이 실시간 손익 추적과 함께 [활성 포지션](/desktop/active-positions) 대시보드에 표시됩니다.
