## A

**AMM (자동화된 마켓 메이커)**
수학적 공식을 사용하여 풀 준비금을 기반으로 토큰 가격을 결정하는 탈중앙화 거래소 프로토콜로, 오더북 없이 토큰 스왑을 가능하게 합니다. PumpSwap, Meteora DAMM V2, Raydium CPMM은 LaserSell이 지원하는 AMM의 예입니다.

**Argon2id**
LaserSell이 패스프레이즈에서 암호화 키를 유도하는 데 사용하는 메모리 하드 비밀번호 해싱 알고리즘입니다. 키스토어에 대한 무차별 대입 공격을 계산적으로 비용이 많이 들게 만듭니다.

**Associated Token Account (ATA)**
주어진 지갑 주소에 대해 특정 SPL 토큰을 보유하는 파생 온체인 계정입니다. 지갑이 처음 토큰을 받을 때 자동으로 생성됩니다.

**Astralane Iris**
레이턴시 최적화를 위한 지역별 엔드포인트를 지원하는 트랜잭션 제출 게이트웨이입니다. LaserSell SDK의 사용 가능한 [전송 대상](/api/transactions/send-targets) 중 하나입니다. Astralane API 키가 필요합니다.

**원자 단위**
토큰의 가장 작은 분할 불가능한 단위입니다. SOL의 경우 원자 단위는 lamport입니다 (1 SOL = 1,000,000,000 lamport). 다른 토큰은 자체 소수점 정밀도를 정의합니다.

## B

**잔액 업데이트**
지갑의 SOL 또는 토큰 잔액이 변경될 때 Exit Intelligence Stream이 보내는 서버 이벤트(`balance_update`)입니다. [서버 이벤트](/api/stream/server-events)를 참조하세요.

**베이시스 포인트 (BPS)**
1/100 퍼센트와 같은 단위입니다. 100 bps = 1%. LaserSell API 전반에서 슬리피지 허용 범위와 파트너 수수료 설정에 사용됩니다.

**본딩 커브**
Pump.fun 같은 런치패드에서 유통 공급량을 기반으로 토큰 가격을 결정하는 데 사용하는 수학적 공식입니다. 본딩 커브에 있는 토큰은 아직 AMM 풀로 마이그레이션되지 않았습니다.

## C

**콜드 스토리지**
장기 보유를 위한 오프라인 지갑 저장소(하드웨어 지갑 등)입니다. 핫 월렛에서 콜드 스토리지로 수익을 정기적으로 이체하세요.

**Configure 메시지**
Exit Intelligence Stream에 연결한 후 보내는 첫 번째 클라이언트 메시지입니다. 지갑 주소와 전략 매개변수를 지정합니다. 서버가 `configured` 이벤트로 응답합니다. [클라이언트 메시지](/api/stream/client-messages)를 참조하세요.

## D

**데드라인 타임아웃**
설정된 시간 내에 다른 전략 조건(목표 수익, 손절매, 트레일링 스탑)이 발동하지 않으면 자동 청산을 강제하는 시간 제한(초)입니다. 정체된 포지션에 자본이 묶이는 것을 방지합니다.

## E

**Exit Intelligence**
LaserSell의 핵심 기술입니다. 지갑 포지션을 모니터링하고, 모든 가격 업데이트에서 전략 조건을 평가하며, 임계값이 충족되면 사전 구축된 서명되지 않은 청산 트랜잭션을 전달하는 실시간 시스템입니다.

**Exit Intelligence Stream**
`wss://stream.lasersell.io/v1/ws`에 대한 지속적인 WebSocket 연결로, 실시간 포지션 추적, 손익 업데이트, 자동 청산 신호를 제공합니다. [Stream 개요](/api/stream/overview)를 참조하세요.

**청산 신호**
전략 조건이 충족될 때 발동되는, 사전 구축된 서명되지 않은 매도 트랜잭션이 포함된 서버 이벤트(`exit_signal_with_tx`)입니다. Exit Intelligence Stream에서 가장 중요한 이벤트입니다. [서버 이벤트](/api/stream/server-events)를 참조하세요.

**ExitApiClient**
`/v1/buy` 및 `/v1/sell`을 통해 요청 시 매수 및 매도 트랜잭션을 빌드하기 위한 REST API 호출을 위한 SDK 클라이언트입니다.

## G

**졸업**
토큰이 본딩 커브(예: Pump.fun)에서 AMM 풀(예: PumpSwap)로 마이그레이션되는 순간입니다. 졸업 시 가격 역학이 크게 변할 수 있습니다.

## H

**`hello_ok`**
Exit Intelligence Stream에 연결한 후 받는 첫 번째 서버 메시지입니다. 세션 ID와 티어 속도 제한이 포함됩니다. [연결 수명 주기](/api/stream/connection-lifecycle)를 참조하세요.

**Helius Sender**
Helius `/fast` 엔드포인트를 통해 최적화된 트랜잭션 랜딩을 위해 라우팅하는 트랜잭션 제출 서비스입니다. API 키가 필요 없습니다. LaserSell의 기본 [전송 대상](/api/transactions/send-targets)입니다.

**핫 월렛**
활발한 거래를 위해 인터넷에 연결된 지갑입니다. LaserSell은 빠른 거래 실행을 위해 기기에 로컬 핫 월렛을 생성합니다.

## J

**Jito 팁**
동일 블록에서 다른 트랜잭션보다 우선순위를 부여하기 위해 Solana 밸리데이터에게 직접 지불하는 것입니다. LaserSell 수수료가 아닌 네트워크 비용입니다.

## K

**키스토어**
LaserSell 클라이언트가 개인키를 로컬에 저장하는 데 사용하는 암호화된 파일 형식입니다. Argon2id로 유도된 키와 함께 XChaCha20-Poly1305 암호화를 사용합니다. [지갑 암호화](/security/wallet-encryption)를 참조하세요.

## L

**Lamport**
SOL의 가장 작은 단위입니다. 1 SOL = 1,000,000,000 lamport. `amount_in_total`, `tip_lamports`, `partner_fee_lamports` 같은 API 필드는 lamport 단위입니다.

**레인 분리**
Exit Intelligence Stream을 두 개의 수신기로 분리하는 기능(TypeScript SDK 전용)입니다: 청산 신호와 포지션 이벤트를 위한 고우선순위 레인과 손익 업데이트를 위한 저우선순위 레인. 빈번한 손익 업데이트가 시간에 민감한 청산 신호를 차단하는 것을 방지합니다.

## M

**마켓 컨텍스트**
풀 주소와 구성을 포함하여 토큰 시장에 사용되는 DEX 풀과 프로토콜을 설명하는 메타데이터입니다. 스트림 이벤트에서 반환되며, 서버 측 해석을 건너뛰기 위해 선택적으로 빌드 요청에 전달됩니다. [마켓 컨텍스트](/api/reference/market-context)를 참조하세요.

**밈코인**
인터넷 문화에서 영감을 받은 암호화폐로, 높은 변동성과 투기적 거래가 특징입니다. LaserSell이 설계된 주요 시장 세그먼트입니다.

**Meteora DBC (Dynamic Bonding Curve)**
초기 출시 단계의 토큰을 위한 Meteora의 본딩 커브 프로토콜입니다. LaserSell이 지원합니다.

**Meteora DAMM V2 (Dynamic AMM V2)**
본딩 커브에서 마이그레이션된 토큰을 위한 Meteora의 동적 자동화된 마켓 메이커입니다. LaserSell이 지원합니다.

**모드 (라우팅)**
`/v1/buy` 및 `/v1/sell` 요청의 `"fast"` 또는 `"secure"` 값을 가진 필드입니다. 빠른 모드는 속도를 최적화하고, 보안 모드는 추가 확인 검사를 추가합니다. 전송 모드와 혼동하지 마세요.

## N

**비수탁형**
사용자가 개인키와 자금에 대한 완전한 제어를 유지하는 아키텍처입니다. LaserSell은 서버에 키를 저장하지 않습니다. 모든 트랜잭션은 서명되지 않은 상태로 반환되며 사용자 또는 애플리케이션이 로컬에서 서명합니다.

## P

**파트너 수수료**
인테그레이터가 매수 또는 매도 트랜잭션에 추가할 수 있는 선택적 수수료로, 온체인 스왑의 일부로 지정된 지갑에 원자적으로 라우팅됩니다. 요청별로 베이시스 포인트 또는 고정 lamport로 설정 가능합니다. Advanced 및 Enterprise 티어에서 이용 가능합니다. [파트너 수수료](/api/reference/partner-fees)를 참조하세요.

**패스프레이즈**
암호화된 키스토어를 잠금 해제하는 데 필요한 사용자 정의 비밀번호입니다. 이 패스프레이즈를 잃으면 지갑을 복호화할 수 없습니다.

**PnL (손익)**
포지션의 현재 예상 수익 또는 손실입니다. Exit Intelligence Stream은 업데이트된 값과 함께 정기적인 간격으로 `pnl_update` 이벤트를 전달합니다.

**`pnl_update`**
추적 중인 포지션의 최신 손익 계산이 포함된 서버 이벤트입니다. 정기적인 간격으로 전달됩니다. 높은 빈도의 전달이 레인 분리가 존재하는 이유입니다. [서버 이벤트](/api/stream/server-events)를 참조하세요.

**포지션 핸들**
`StreamSession` 이벤트가 반환하는 참조 객체로, 추적 중인 포지션을 ID, 토큰 계정, 민트, 지갑으로 식별합니다. 포지션 닫기 및 청산 신호 요청에 사용됩니다.

**`position_opened`**
Exit Intelligence Stream이 추적 중인 지갑에서 새로운 토큰 매수를 감지할 때 발동되는 서버 이벤트입니다. 포지션 ID, 민트, 진입 비용, 마켓 컨텍스트가 포함됩니다. [서버 이벤트](/api/stream/server-events)를 참조하세요.

**`position_closed`**
추적 중인 포지션이 완전히 청산되거나 제거될 때 발동되는 서버 이벤트입니다. [서버 이벤트](/api/stream/server-events)를 참조하세요.

**우선 수수료**
연산 우선순위를 위해 밸리데이터에게 지불하는 표준 Solana 블록체인 수수료입니다. Jito 팁과는 별도입니다.

**개인키**
지갑 자산에 대한 제어를 부여하는 암호화 키입니다. LaserSell 클라이언트는 이를 로컬에서 암호화하며 절대 전송하지 않습니다. API와 SDK는 개인키를 요구하지 않으며, 모든 트랜잭션은 서명되지 않은 상태로 반환됩니다.

**Pump.fun**
토큰이 AMM으로 졸업하기 전에 본딩 커브에서 거래를 시작하는 Solana 토큰 런치패드입니다.

**PumpSwap**
Pump.fun 토큰이 본딩 커브에서 졸업한 후 마이그레이션되는 AMM입니다. LaserSell이 지원합니다.

## R

**속도 제한**
가격 티어에 따라 사용할 수 있는 요청 또는 리소스의 상한입니다. REST API(분당 요청 수)와 Exit Intelligence Stream(지갑, 포지션, 세션) 모두에 적용됩니다. [속도 제한 및 티어](/api/reference/rate-limits)를 참조하세요.

**Raydium CPMM (Constant Product Market Maker)**
Raydium의 표준 AMM 풀입니다. LaserSell이 지원합니다.

**Raydium Launchpad**
Raydium의 토큰 출시 플랫폼입니다. LaserSell이 지원합니다.

**라우팅 수수료**
LaserSell Router를 통해 실행되는 모든 주문에 적용되는 거래당 수수료입니다. 구독 티어에 따라 비율이 달라집니다: 2%(무료), 0.9%(Professional), 0.5%(Advanced) 또는 맞춤(Enterprise). 온체인에서 원자적으로 적용됩니다. [수수료 구조](/pricing/fee-structure)를 참조하세요.

**RPC (Remote Procedure Call)**
Solana 블록체인에 대한 게이트웨이 역할을 하는 노드입니다. LaserSell은 체인 데이터를 읽고 트랜잭션을 전송하기 위해 RPC 연결을 사용합니다. 프로덕션 사용에는 전용 RPC 엔드포인트를 권장합니다.

**러그 풀**
토큰의 유동성이 철회되거나 매도가 비활성화되어 가격이 폭락하는 시나리오입니다. LaserSell의 손절매, 트레일링 스탑, 졸업 시 매도 기능이 노출을 제한하는 데 도움이 됩니다.

## S

**시드 구문**
지갑을 생성하거나 복원하는 데 사용되는 12 또는 24개의 니모닉 단어 구문입니다. 초기 지갑 생성 시 한 번만 표시됩니다.

**졸업 시 매도**
토큰이 본딩 커브에서 AMM 풀로 마이그레이션되는 순간 포지션을 자동으로 청산하는 전략 조건입니다. 졸업 후 덤프로부터 보호합니다.

**전송 모드**
빌드 요청의 필드(`"rpc"`, `"helius_sender"` 또는 `"astralane"`)로, 서버에게 어떤 제출 컨텍스트에 맞게 트랜잭션을 최적화할지 알려줍니다. 선택한 전송 대상과 일치해야 합니다.

**전송 대상**
SDK에게 서명된 트랜잭션을 어디에 제출할지 알려주는 구성입니다. 옵션: Default RPC, Custom RPC, Helius Sender, Astralane Iris. [전송 대상](/api/transactions/send-targets)을 참조하세요.

**슬리피지**
거래의 예상 가격과 실제 실행 가격의 차이입니다. 높은 변동성과 낮은 유동성이 슬리피지를 증가시킵니다.

**손절매**
가격이 진입 가격에서 설정된 비율만큼 하락할 때 포지션을 자동으로 매도하는 전략 조건입니다. 단일 거래에서의 하락을 제한합니다.

**StreamClient**
Exit Intelligence Stream을 위한 저수준 SDK 클라이언트입니다. WebSocket 연결, 재연결, 메시지 프레이밍을 관리합니다. 원시 서버 메시지를 반환합니다. [연결 수명 주기](/api/stream/connection-lifecycle)를 참조하세요.

**StreamSession**
`StreamClient`의 고수준 SDK 래퍼입니다. 자동 포지션 추적, 데드라인 타이머, 포지션 핸들이 포함된 타입 이벤트, 전략 업데이트 메서드를 제공합니다. 대부분의 통합에 권장됩니다.

## T

**목표 수익**
포지션의 손익이 설정된 수익 비율에 도달할 때 자동 매도를 트리거하는 전략 조건입니다.

**트레일링 스탑**
포지션이 열린 이후 관찰된 최고 수익을 추적하는 동적 손절매입니다. 수익이 그 최고점에서 설정된 비율만큼 하락하면 청산 신호가 발동됩니다. 반전되기 전에 상승하는 포지션에서 수익을 확보합니다.

## U

**USD1**
매도 트랜잭션에서 SOL 대신 대체 출력 자산으로 지원되는 스테이블코인입니다.

**서명되지 않은 트랜잭션**
LaserSell API가 반환하는 base64로 인코딩된 Solana `VersionedTransaction`입니다. 제출 전에 개인키로 로컬에서 서명해야 합니다. LaserSell은 서명을 처리하지 않습니다.

## V

**VersionedTransaction**
버전이 지정된 인스트럭션 레이아웃과 주소 조회 테이블을 지원하는 Solana 트랜잭션 형식입니다. LaserSell API가 반환하는 모든 트랜잭션은 이 형식을 사용합니다.

## W

**WebSocket (WSS)**
실시간 양방향 데이터 스트림을 가능하게 하는 통신 프로토콜입니다. Exit Intelligence Stream은 최소한의 레이턴시로 포지션 이벤트와 청산 신호를 전달하기 위해 WSS를 사용합니다.

## X

**XChaCha20-Poly1305**
LaserSell 클라이언트가 키스토어 파일 내에서 개인키를 보호하는 데 사용하는 인증 암호화 알고리즘입니다. 기밀성과 무결성 검증을 모두 제공합니다.
