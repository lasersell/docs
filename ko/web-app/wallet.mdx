Wallet 페이지에서 웹 트레이딩에 사용되는 지갑을 관리할 수 있습니다. 모든 지갑은 Privy의 Trusted Execution Environment(TEE)로 보호됩니다.

## 임베디드 지갑

처음 로그인하면 Privy가 자동으로 임베디드 Solana 지갑을 생성합니다. 이 지갑은 Wallet 페이지에 "LaserSell Wallet"으로 표시되며 즉시 사용할 수 있습니다.

임베디드 지갑은 브라우저가 아닌 계정 ID에 연결되어 있습니다. LaserSell 계정으로 로그인하면 어떤 기기에서든 동일한 지갑에 접근할 수 있습니다.

## 지갑 제한

각 계정은 최대 **2개의 웹 지갑**(생성 또는 가져오기)을 보유할 수 있습니다. 여기에는 온보딩 시 생성된 임베디드 지갑과 가져온 모든 지갑이 포함됩니다. 다른 지갑을 추가하려면 먼저 기존 지갑을 제거하세요.

## 지갑 가져오기

기존 Solana 지갑을 가져와 웹 트레이딩에 사용할 수 있습니다.

<Steps>
  <Step title="Import Wallet 클릭">
    Wallet 페이지에서 "Import Wallet" 버튼을 클릭하세요.
  </Step>
  <Step title="개인키 입력">
    base58로 인코딩된 개인키를 붙여넣으세요. 키는 브라우저에서 Privy의 TEE로 직접 전송됩니다. LaserSell 서버를 경유하지 않습니다.
  </Step>
  <Step title="지갑 등록 완료">
    LaserSell이 지갑으로 증명 메시지에 서명하여 소유권을 확인한 다음 거래용으로 등록합니다.
  </Step>
</Steps>

<Info>
  개인키는 암호화되어 Privy의 Trusted Execution Environment에 저장됩니다. [Privy의 보안에 대해 알아보기](https://docs.privy.io/guide/security).
</Info>

## SOL 입금

거래를 하려면 지갑에 토큰 매수와 트랜잭션 수수료 지불을 위한 SOL이 필요합니다.

1. Wallet 페이지로 이동
2. 지갑에서 **Copy Address**를 클릭
3. 외부 지갑(Phantom, Backpack 등)에서 해당 주소로 SOL을 전송

## SOL 출금

아무 지갑에서나 **Withdraw**를 클릭하여 SOL 또는 토큰을 외부 주소로 전송할 수 있습니다. 출금은 Privy에 의해 서명되고 LaserSell의 Solana RPC를 통해 제출됩니다.

## 개인키 내보내기

임베디드 지갑의 경우 언제든지 개인키를 내보낼 수 있습니다.

1. 지갑 카드에서 **Export Key**를 클릭
2. Privy가 개인키를 표시하는 보안 창을 엽니다
3. 복사하여 안전하게 보관하세요

내보내기 과정에서 LaserSell은 개인키를 볼 수 없습니다. 전체 프로세스는 Privy의 보안 iframe 내에서 이루어집니다.

## 지갑 제거

가져온 지갑을 제거하려면 지갑 카드의 휴지통 아이콘을 클릭하세요. 이렇게 하면 LaserSell에서 지갑이 제거되고 Privy의 TEE에서 개인키가 영구적으로 삭제됩니다.

임베디드 지갑은 Privy 계정에 연결되어 있으므로 대시보드에서 직접 제거할 수 없습니다. 임베디드 지갑을 완전히 제거하려면 먼저 개인키를 내보낸 다음 고객 지원에 연락하여 계정 삭제를 요청하세요.

<Warning>
  지갑 제거는 되돌릴 수 없습니다. 자금에 대한 접근을 유지하려면 제거 전에 개인키를 내보내세요.
</Warning>

## 활성 지갑

여러 지갑이 있는 경우 지갑 카드에서 **Set Active**를 클릭하여 활성 지갑을 설정할 수 있습니다. 활성 지갑은 파란색으로 강조 표시되며 새 거래 세션의 기본값으로 사용됩니다.

## 지갑 보안

| 속성 | 세부사항 |
|----------|--------|
| **키 저장** | Privy TEE (하드웨어 격리 보안 영역) |
| **키 접근** | LaserSell은 Privy에 서명을 요청할 수 있지만 키를 직접 보지 못함 |
| **지갑 증명** | 등록 시 ed25519 서명으로 소유권 증명 |
| **크로스 디바이스** | 모든 브라우저와 기기에서 동일한 지갑 사용 |
| **내보내기** | Privy의 보안 UI를 통해 언제든지 개인키 내보내기 가능 |
