Trading 페이지는 웹 트레이딩의 메인 제어 센터입니다. **Positions** 탭에서 활성 거래를 모니터링하고, **Copy Trading** 탭에서 다른 지갑을 미러링할 수 있습니다.

## 시작하기

아직 웹 트레이딩 세션을 시작하지 않았다면 Trading 페이지에 설정 양식이 표시됩니다:

1. 모니터링할 **지갑을 선택**합니다 (등록된 지갑 중에서)
2. **청산 전략을 설정**합니다 (목표 수익, 손절매, 트레일링 스탑, 최대 슬리피지)
3. **Start Web Trading**을 클릭합니다

시작되면 LaserSell이 지갑을 모니터링하고 전략 임계값에 도달할 때 자동으로 거래를 실행합니다.

## Positions 탭

Positions 탭은 세 열 레이아웃을 사용합니다:

- **왼쪽**: 손익 요약이 포함된 전체 오픈 포지션 목록
- **가운데**: 선택한 포지션의 상세 보기 (토큰 정보, 진입 가격, 현재 수익, 매도 버튼)
- **오른쪽**: 청산 전략 설정 (언제든지 편집 가능)

### 청산 전략 설정

| 설정 | 설명 |
|---------|-------------|
| **목표 수익 %** | 수익이 해당 비율에 도달하면 매도 |
| **손절매 %** | 손실이 해당 비율에 도달하면 매도 |
| **트레일링 스탑 %** | 가격을 해당 비율로 추적하여 수익을 확보 |
| **최대 슬리피지 %** | 자동 매도 시 허용 가능한 최대 슬리피지 (기본 5%) |
| **졸업 시 매도** | 토큰이 pump.fun에서 Raydium으로 졸업할 때 자동 매도 |

변경 사항은 모든 활성 포지션에 즉시 적용됩니다.

### 수동 매도

포지션에서 **Sell Now**를 클릭하면 전략 임계값에 관계없이 즉시 매도가 실행됩니다.

## Copy Trading 탭

카피 트레이딩을 사용하면 다른 지갑의 거래를 자동으로 미러링할 수 있습니다.

### 감시 지갑 추가

1. **Add Watch Wallet**을 클릭
2. Solana 지갑 주소 또는 `.sol` 도메인 이름(SNS를 통해 확인)을 입력
3. 닉네임과 자동 매수 금액(SOL 단위)을 설정
4. 감시 목록에 지갑이 추가됩니다

감시 중인 지갑이 토큰을 매수하면, LaserSell이 사용자의 지갑으로 동일한 매수를 자동 실행합니다.

### 미러 설정

| 설정 | 설명 |
|---------|-------------|
| **지갑당 최대 포지션** | 동시 미러 포지션 제한 |
| **쿨다운** | 미러 매수 간 최소 시간 |
| **크리에이터 토큰 건너뛰기** | 감시 지갑이 생성한 토큰 무시 |
| **최대 활성 SOL** | 미러 포지션에 투입되는 총 SOL 상한 |
| **매수 슬리피지** | 미러 매수 시 최대 슬리피지 |
| **최소 풀 유동성** | 유동성이 낮은 토큰 건너뛰기 |
| **최대 진입 드리프트** | 감시 지갑의 진입가에서 가격이 너무 벗어나면 건너뛰기 |
| **최대 연속 손실** | N회 연속 손실 후 감시 지갑 자동 비활성화 |

## 세션 제어

헤더의 **에이전트 토글**(전원 아이콘)을 사용하여 웹 트레이딩을 시작하거나 중지합니다. 상태 점은 다음을 나타냅니다:

- 🟢 **초록색**: 활성 및 모니터링 중
- 🔵 **파란색**: 새 포지션 감시 중
- 🟡 **노란색**: 일시 중지됨 (오류 또는 비활성)
- ⚪ **회색**: 중지됨

<Warning>
  데스크톱 앱과 웹 앱 모두에서 동일한 지갑을 모니터링하고 있다면, 임계값 도달 시 두 앱 모두 매도를 시도합니다. 하나로 전환할 때 다른 하나를 꺼주세요.
</Warning>
