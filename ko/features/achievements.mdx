LaserSell은 거래 마일스톤을 추적하고 달성 시 업적을 수여합니다. 10개 카테고리에 걸쳐 85개 이상의 업적이 있으며, 누구나 달성할 수 있는 Common 마일스톤부터 탁월한 성과가 필요한 Legendary 업적까지 다양합니다.

<Note>
잠금 해제된 업적은 트레이더에게 이전에는 제공된 적 없는 보상으로 교환할 수 있게 될 것입니다. LaserSell 토큰은 없으며 에어드롭도 없을 것입니다.
</Note>

## 업적 작동 방식

업적은 매시간 주기로 자동으로 확인됩니다. LaserSell 프로토콜을 통해 라우팅된 거래만 업적 진행에 반영됩니다. 새로운 업적을 잠금 해제하면 앱이 알림을 보냅니다.

숨겨진 업적은 잠금 해제할 때까지 조건이 공개되지 않습니다.

## 희귀도 티어

| 희귀도 | 설명 |
| :--- | :--- |
| **Common** | 입문 수준의 마일스톤. 거래를 시작하면 자연스럽게 잠금 해제됩니다. |
| **Uncommon** | 지속적인 거래 활동이 필요한 중간 수준의 성취. |
| **Rare** | 진정한 헌신이나 기술이 필요한 중요한 업적. |
| **Epic** | 소수의 트레이더만이 달성하는 탁월한 마일스톤. |
| **Legendary** | 시스템에서 가장 희귀한 업적. 엄청난 거래량, 일관성, 또는 단 한 번의 탁월한 성과가 필요합니다. |

## 카테고리

### 거래량

누적 거래량 마일스톤. LaserSell을 통해 더 많이 거래할수록 더 높이 올라갑니다.

### 거래 수

첫 번째 거래부터 수천 건까지의 총 완료 거래 마일스톤.

### 연속 거래

연속 거래일 마일스톤. LaserSell을 통해 라우팅된 거래를 최소 한 건 완료하면 연속 거래일이 카운트됩니다. 하루를 놓치면 현재 연속 기록이 초기화됩니다.

현재 연속 기록과 최장 연속 기록은 앱의 업적 탭 상단에 표시됩니다.

### 수익

SOL로 측정된 누적 실현 수익 마일스톤.

### 속도

짧은 시간 프레임에서 수익을 낸 거래에 대한 업적. 빠른 실행과 신속한 청산에 대한 보상입니다.

### 단일 거래

단일 포지션의 성과에 기반한 업적. 견실한 수익률부터 탁월한 배수까지.

### 승률

최소 거래 수 요건이 있는 승률 마일스톤으로 어뷰징을 방지합니다. 수백 건의 거래에 걸쳐 높은 승률을 유지하는 것은 가장 어려운 일 중 하나입니다.

### 미러

[미러 트레이딩](/desktop/mirror-trading) 업적. 감시 지갑 전반에 걸친 카피 트레이딩 활동과 성과를 추적합니다.

### 시크릿

잠금 해제할 때까지 조건이 공개되지 않는 숨겨진 업적. 거래하면서 자연스럽게 발견하게 됩니다.

### 마스터리

카테고리의 모든 업적을 완료하면 수여되는 메타 업적. 각 카테고리마다 하나의 마스터리 업적이 있습니다. 모든 시크릿 업적을 완료하면 유일한 Legendary 티어 마스터리를 획득합니다.

## 연속 기록

거래 연속 기록은 LaserSell을 통해 라우팅된 거래를 최소 한 건 완료한 연속 일수를 추적합니다.

- **현재 연속 기록** — 오늘을 포함하여 연속으로 거래한 일수.
- **최장 연속 기록** — 역대 최고 기록.
- **마지막 거래 날짜** — 가장 최근 거래가 기록된 시점.

하루를 놓치면 현재 연속 기록이 0으로 초기화됩니다. 최장 연속 기록은 영구적으로 보존됩니다.

## API 접근

업적은 프로그래밍 방식으로 이용할 수 있습니다:

```
GET /v1/achievements
```

모든 업적의 잠금 해제 상태와 타임스탬프를 반환합니다. 시크릿 카테고리의 잠긴 업적은 마스킹된 설명을 반환합니다.
