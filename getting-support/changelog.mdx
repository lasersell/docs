### v0.2.0 - January 28, 2026

#### **üöÄ New Features & Protocol Support**

- **Expanded DEX Support:** Full integration for DAMM v2, CPMM, PumpSwap, and [Bonk.fun](http://Bonk.fun).
- **Launchpad Support:** Comprehensive support for all Raydium Launchpad coins (including USD1 & SOL tokens).
- **Cross-Platform:** Added official support for macOS and Windows.
- **Security Tools:** New CLI command `--export-private-key [path]` for easier key management.
- **LaserSell is now FREE to download**: No more access passes and no monthly subscriptions.

#### **üìà Trading & Logic Improvements**

- **Smart Cost Basis:** Updated tracking logic to calculate counts and cost basis correctly across multiple buys.
- **Simplified Strategies:** Consolidated Take Profit (TP) and Stop Loss (SL) strategies into a streamlined, percentage-only format.
- **Flexible Timeouts:** You can now disable timeouts by setting the value to 0.
- **Migration Support:** Added foundational support for token migrations.

#### **üõ† UI & UX Enhancements**

- **UI Redesign:** A fresh look and feel for a better user experience.
- **Improved Onboarding:** Adjusted the first-time user flow for a smoother setup.
- **Smart Keystores:** The system will no longer generate a new keystore if an existing one is detected, preventing accidental overwrites.

#### **üîß CLI & Configuration Changes**

- **Option Consolidation:** Removed `-c` and `-l` flags. These have been merged for simplicity (Note: -m remains unchanged).
- **Stream Cleanup:** Removed Helius Laserstream support. The config stream has been renamed to websocket for clarity.

#### **üêû Bug Fixes**

- **PnL Accuracy:** Fixed an issue where manual buys would interfere with PnL calculations and accidentally trigger autosell.
- **Tracking Fix:** Resolved a bug where secondary buys of the same coin were not being tracked by LaserSell.
- **Zombie Websockets:** Fixed a bug preventing mint detection on the free tier websocket.
