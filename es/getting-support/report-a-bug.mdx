## Vulnerabilidades de seguridad

Si descubres una vulnerabilidad de seguridad en cualquier producto de LaserSell, por favor repórtala de manera responsable:

- **Correo:** [security@lasersell.io](mailto:security@lasersell.io)
- **PGP:** Te alentamos a cifrar tu reporte con nuestra [clave PGP](https://www.lasersell.io/lasersell-pgp-key.asc). Las guías completas de divulgación están en [www.lasersell.io/.well-known/security.txt](https://www.lasersell.io/.well-known/security.txt).

**No** abras un issue público de GitHub para vulnerabilidades de seguridad.

## Errores de software

Para errores en LaserSell o el SDK de LaserSell, usa uno de estos canales:

- **Issues de GitHub:**
  - [lasersell/lasersell](https://github.com/lasersell/lasersell/issues) (Cliente)
  - [lasersell/lasersell-sdk](https://github.com/lasersell/lasersell-sdk/issues) (SDK)
- **Ticket de Discord:** Abre un ticket de soporte en [discord.gg/lasersell](https://discord.gg/lasersell)

Al reportar, incluye la mayor cantidad de detalles posible: pasos para reproducir, mensajes de error, tu sistema operativo/plataforma y la versión (`lasersell --version`).

### Proceso de resolución

Nuestro objetivo es clasificar y corregir los errores confirmados con prontitud.

- **Notificación:** Una vez implementada una corrección, te notificaremos para que puedas obtener la última actualización.
- **Actualizaciones:** Consulta el canal de Telegram [LaserSell Announcements](https://t.me/lasersell) o el canal `#announcements` de Discord para verificar que la corrección está incluida en tu versión.
